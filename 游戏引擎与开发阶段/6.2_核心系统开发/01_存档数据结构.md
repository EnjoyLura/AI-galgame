## 2. 数据字段说明

### 2.1 meta

| 字段 | 类型 | 说明 |
|------|------|------|
| save_id | string | 存档唯一标识 |
| timestamp | datetime | 保存时间 |
| play_count | int | 游玩次数 |
| play_time | int | 累计游玩时间（秒） |

### 2.2 variables

| 字段组 | 说明 |
|--------|------|
| route | 路线状态：当前章节、路线、季节、游戏日 |
| yuma | 悠真的成长参数：growth（成长阶段字符串）、courage（是否勇敢选择）、belonging（是否找到归属） |

### 2.3 flags

| 字段组 | 说明 |
|--------|------|
| common | 共通线标志位 |
| *_route | 各个人线专属标志位 |

### 2.4 endings

| 字段 | 说明 |
|------|------|
| *_good | 各角色Good End达成标记 |
| *_bad | 各角色Bad End达成标记 |
| outsider | 局外人结局（烟花大会选择独自看烟花） |
| true_end | True End达成标记 |
