## 2. Flag系统与进入个人线条件

### 2.1 角色Flag定义

| Flag名称          | 说明        | 获取条件                                        |
| --------------- | --------- | ------------------------------------------- |
| `FLAG_AYANE`    | 绫音线准入Flag | 加入学生会 **AND** 学园祭帮助绫音 **AND** 在转折剧情中解决画画问题  |
| `FLAG_CHINATSU` | 千夏线准入Flag | 学园祭为千夏加油 **AND** 一起去田径联赛 **AND** 去医院探病照顾    |
| `FLAG_CHISA`    | 千纱线准入Flag | 当图书委员助手 **AND** 学园祭帮助千纱做校史宣传 **AND** 去男主家做客 |
| `FLAG_AKANE`    | 茜线准入Flag  | 天台搭话约定一起吃便当 **AND** 周末去便利店3次 **AND** 和茜一起打工 |
| `FLAG_KOHARU`   | 小春线准入Flag | 周末在家不出门累计2次及以上（无转折线）                        |

### 2.2 进入个人线条件（烟花大会分歧）

```
if FLAG_AYANE == true AND 选择绫音:
    → 绫音线
else if FLAG_CHINATSU == true AND 选择千夏:
    → 千夏线
else if FLAG_CHISA == true AND 选择千纱:
    → 千纱线
else if FLAG_AKANE == true AND 选择茜:
    → 茜线
else if (FLAG_KOHARU >= 2) OR (所有角色FLAG均为false AND FLAG_KOHARU >= 1):
    → 小春线
else:
    → Bad End（单身线）
```

## 3. 选择点设计原则

### 3.1 多样化的选择场景

| 场景类型        | 触发时机   | 示例          |
| ----------- | ------ | ----------- |
| **午休选择**    | 中午休息时间 | 去哪里吃午餐/度过午休 |
| **放学后选择**   | 放学时    | 去哪里/和谁一起    |
| **周末选择**    | 周六或周日  | 出门还是在家      |
| **剧情中选择**   | 特定剧情中  | 如何回应/如何行动   |
| **电话/消息选择** | 收到联系时  | 回复谁/去哪里     |

### 3.2 动态选项系统

选项不是固定的，会根据**已发生的事件**、**当前地点**、**时间**动态变化：

- 加入学生会后，选项中会出现"去学生会室"
- 成为图书委员助手后，选项中会出现"去图书馆整理书架"
- 发现画室后，选项中会出现"去画室"
- 某个角色剧情推进后，该角色的选项会变化

| 绫音 | 河边，绫音穿浴衣，说"谢谢你"  | → 绫音个人线 |
| 千夏 | 河边，千夏笑容灿烂但手在发抖   | → 千夏个人线 |
| 千纱 | 樱花隧道附近，千纱穿白色浴衣   | → 千纱个人线 |
| 茜  | 河边，茜说真的可以吗       | → 茜个人线  |
| 小春 | 河边，小春说"就今晚，不是妹妹" | → 小春个人线 |

> **Bad End（单身线）**：悠真想不出可以邀请谁，决定不去烟花大会。独白：「烟花大会啊...算了，还是继续做一个旁观者吧。」第一学期结束，什么也没有改变。

## 8. 动态选项系统详解

### 8.1 选项根据社团状态变化

| 社团状态    | 新增选项       | 说明      |
| ------- | ---------- | ------- |
| 加入学生会   | "去学生会室"    | 绫音相关剧情  |
| 当图书委员助手 | "去图书馆整理书架" | 千纱相关剧情  |
| 什么都没加入  | "随便逛逛"     | 可能发现新地点 |
| 发现画室后   | "去画室"      | 绫音深度剧情  |
| 接受打工后   | "去打工"      | 茜相关剧情   |

### 8.2 选项根据角色关系变化

| 关系阶段 | 选项变化   | 说明       |
| ---- | ------ | -------- |
| 初遇   | 基础选项   | 普通对话     |
| 熟悉   | 新增亲密选项 | 可以开玩笑/关心 |
| 深入   | 新增特殊选项 | 可以触碰敏感话题 |
| 转折后  | 新增专属选项 | 角色专属剧情   |

### 8.3 避免重复的设计

- **场景多样化**：教室、学生会室、图书馆、操场、天台、便利店、家里、商店街等
- **时机多样化**：午休、放学后、周末、假期、特定事件前后
- **选项内容多样化**：不是简单的"去A/去B/去C"，而是有剧情意义的"关心/调侃/沉默"

## 9. 内容解耦说明

### 9.1 各角色剧情线独立性

| 角色     | 进入个人线条件（与2.1 Flag定义一致） | 与其他角色的交集 |
| ------ | -------------------------- | -------- |
| **绫音** | 加入学生会 **AND** 学园祭帮助绫音 **AND** 转折剧情中解决画画问题 | 无互斥，独立条件 |
| **千夏** | 学园祭为千夏加油 **AND** 一起去田径联赛 **AND** 去医院探病照顾 | 无互斥，独立条件 |
| **千纱** | 当图书委员助手 **AND** 学园祭帮助千纱做校史宣传 **AND** 去男主家做客 | 无互斥，独立条件 |
| **茜**  | 天台搭话约定一起吃便当 **AND** 周末去便利店3次 **AND** 和茜一起打工 | 无互斥，独立条件 |
| **小春** | 周末在家不出门累计2次及以上（无转折线） | 无互斥，独立条件 |

### 9.2 互斥事件说明

- **转折剧情期间**：进入角色转折线后无法跳出，直到该角色转折剧情结束
- **学园祭**：是部分角色Flag的必要条件之一（绫音、千夏、千纱）
- **学习会**：不影响进入个人线的条件，只是增加好感度
- **其他时间**：各角色剧情完全独立，互不影响

### 9.3 项目结构

本场景表只定义**共通线的框架和选择点**，不涉及：

- 个人线剧情（在个人线剧本中定义）
- 具体对话文本（在剧本中定义）
- CG/BGM需求（在美术/音频文档中定义）
