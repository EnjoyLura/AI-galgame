## 2. Flag系统与进入个人线条件

### 2.1 角色Flag定义

| Flag名称          | 说明        | 获取条件                                        |
| --------------- | --------- | ------------------------------------------- |
| `FLAG_AYANE`    | 绫音线准入Flag | 加入学生会 **AND** 学园祭帮助绫音 **AND** 在转折剧情中解决画画问题  |
| `FLAG_CHINATSU` | 千夏线准入Flag | 学园祭为千夏加油 **AND** 一起去田径联赛 **AND** 去医院探病照顾    |
| `FLAG_CHISA`    | 千纱线准入Flag | 当图书委员助手 **AND** 学园祭帮助千纱做校史宣传 **AND** 去男主家做客 |
| `FLAG_AKANE`    | 茜线准入Flag  | 天台搭话约定一起吃便当 **AND** 周末去便利店3次 **AND** 和茜一起打工 |
| `FLAG_KOHARU`   | 小春线准入Flag | 周末在家不出门累计2次及以上（无转折线）                        |

### 2.2 进入个人线条件（烟花大会分歧）

```
if FLAG_AYANE == true AND 选择绫音:
    → 绫音线
else if FLAG_CHINATSU == true AND 选择千夏:
    → 千夏线
else if FLAG_CHISA == true AND 选择千纱:
    → 千纱线
else if FLAG_AKANE == true AND 选择茜:
    → 茜线
else if (FLAG_KOHARU >= 2) OR (所有角色FLAG均为false AND FLAG_KOHARU >= 1):
    → 小春线
else:
    → Bad End（单身线）
```

## 3. 选择点设计原则

### 3.1 多样化的选择场景

| 场景类型        | 触发时机   | 示例          |
| ----------- | ------ | ----------- |
| **午休选择**    | 中午休息时间 | 去哪里吃午餐/度过午休 |
| **放学后选择**   | 放学时    | 去哪里/和谁一起    |
| **周末选择**    | 周六或周日  | 出门还是在家      |
| **剧情中选择**   | 特定剧情中  | 如何回应/如何行动   |
| **电话/消息选择** | 收到联系时  | 回复谁/去哪里     |

### 3.2 动态选项系统

选项不是固定的，会根据**已发生的事件**、**当前地点**、**时间**动态变化：

- 加入学生会后，选项中会出现"去学生会室"
- 成为图书委员助手后，选项中会出现"去图书馆整理书架"
- 发现画室后，选项中会出现"去画室"
- 某个角色剧情推进后，该角色的选项会变化

| 千纱 | 河边 | 悠真带千纱来到樱花隧道附近会和后。千纱穿着白色浴衣，温柔又美丽。烟花开始，照亮樱花树。千纱看着悠真："悠真，你还记得七年前的约定吗？"她的眼睛里映着烟花："我们约定做一辈子的好朋友……"悠真点头。千纱微笑小声说："现在，我不只是想做朋友了……"悠真说你刚刚说了什么？，千纱红着脸说没什么没什么                    | → 千纱个人线 |
| 茜  | 河边 | 悠真在河边等茜。茜穿着简单的浴衣，但依旧可爱。她有些紧张："悠真……我真的可以来吗？"悠真点头。烟花开始，茜看着烟花，眼睛亮晶晶的。她小声说："我这是第一次来烟花大会……和别人一起。"她看着悠真："谢谢你，悠真。"                                                           | → 茜个人线  |
| 小春 | 河边 | 悠真和小春一起去烟花大会。小春穿着可爱的浴衣，蹦蹦跳跳很开心。她像小时候一样牵着悠真的手，但走到河边，她停下来，看着悠真，表情有些认真："哥哥……"小春深吸一口气，她的脸颊通红，眼神里是悠真从未见过的认真。悠真问怎么啦，小春说没什么，今晚的烟花很美                                          | → 小春个人线 |

***

> **Bad End（单身线）**：
> 悠真想不出可以邀请谁，或者没有满足任何角色的FLAG。他躺在床上，看着天花板，叹了口气。独白：「烟花大会啊...算了，还是继续做一个旁观者吧。」
> 烟花大会当天，悠真一个人在房间里，听着外面烟花的声音，心里有些失落。第一学期结束，什么也没有改变。
> 悠真看着窗外，心里有些后悔——如果当初勇敢一点，是不是会不一样？但世上没有后悔药。这个学期就这样结束了。

| 小春      | 小春线  | 兄妹关系、家族秘密、情感抉择 |

## 8. 动态选项系统详解

### 8.1 选项根据社团状态变化

| 社团状态    | 新增选项       | 说明      |
| ------- | ---------- | ------- |
| 加入学生会   | "去学生会室"    | 绫音相关剧情  |
| 当图书委员助手 | "去图书馆整理书架" | 千纱相关剧情  |
| 什么都没加入  | "随便逛逛"     | 可能发现新地点 |
| 发现画室后   | "去画室"      | 绫音深度剧情  |
| 接受打工后   | "去打工"      | 茜相关剧情   |

### 8.2 选项根据角色关系变化

| 关系阶段 | 选项变化   | 说明       |
| ---- | ------ | -------- |
| 初遇   | 基础选项   | 普通对话     |
| 熟悉   | 新增亲密选项 | 可以开玩笑/关心 |
| 深入   | 新增特殊选项 | 可以触碰敏感话题 |
| 转折后  | 新增专属选项 | 角色专属剧情   |

### 8.3 避免重复的设计

- **场景多样化**：教室、学生会室、图书馆、操场、天台、便利店、家里、商店街等
- **时机多样化**：午休、放学后、周末、假期、特定事件前后
- **选项内容多样化**：不是简单的"去A/去B/去C"，而是有剧情意义的"关心/调侃/沉默"

## 9. 内容解耦说明

### 9.1 各角色剧情线独立性

| 角色     | 进入个人线条件（与2.1 Flag定义一致）                      | 与其他角色的交集 |
| ------ | ------------------------------------------- | -------- |
| **绫音** | 加入学生会 **AND** 学园祭帮助绫音 **AND** 转折剧情中解决画画问题   | 无互斥，独立条件 |
| **千夏** | 学园祭为千夏加油 **AND** 一起去田径联赛 **AND** 去医院探病照顾    | 无互斥，独立条件 |
| **千纱** | 当图书委员助手 **AND** 学园祭帮助千纱做校史宣传 **AND** 去男主家做客 | 无互斥，独立条件 |
| **茜**  | 天台搭话约定一起吃便当 **AND** 周末去便利店3次 **AND** 和茜一起打工 | 无互斥，独立条件 |
| **小春** | 周末在家不出门累计2次及以上（无转折线）                        | 无互斥，独立条件 |

### 9.2 互斥事件说明

- **转折剧情期间**：进入角色转折线后无法跳出，直到该角色转折剧情结束
- **学园祭**：是部分角色Flag的必要条件之一（绫音、千夏、千纱）
- **学习会**：不影响进入个人线的条件，只是增加好感度
- **其他时间**：各角色剧情完全独立，互不影响

### 9.3 项目结构

本场景表只定义**共通线的框架和选择点**，不涉及：

- 个人线剧情（在个人线剧本中定义）
- 具体对话文本（在剧本中定义）
- CG/BGM需求（在美术/音频文档中定义）
