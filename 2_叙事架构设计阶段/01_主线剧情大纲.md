# 《君と描くパレット》— 主线剧情大纲

***

## 1. 五路线剧情总览

### 1.1 共通线时间轴整合

```
4月 ──────────── 5月 ──────────── 6月 ──────── 7月
                  第一学期
│                │              │          │
├─ 入学式         ├─ 学园祭        ├─ 期中考试   ├─ 烟花大会
├─ 各线初遇        ├─ 关系深化      ├─ 转折事件   ├─ 共通线结束
├─ 社团/日常       ├─ 田径联赛(千夏)  ├─ 学习会    ├─ 个人线开始
│                │              │          ├─ 个人线发展
│                │              │          ├─ 个人线高潮
│                │              │          └─ 结局/特殊CG
```

> **期中考试学习会**安排在6月中旬，作为共通线中全员参与的强制事件。玩家可选择对应角色的家和大家一起去复习，每个角色的复习场景展现其性格侧面，同时也是好感flag的积累节点。

> **烟花大会**是共通线与个人线的分界点。若玩家在共通线中未与任何角色建立足够深入的关系，烟花大会将触发 **Bad End（单身线）**——悠真放弃去看烟花，第一学期在"无色"中结束。
