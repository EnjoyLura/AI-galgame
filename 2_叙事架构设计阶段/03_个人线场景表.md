# 《君と描くパレット》— 个人线场景表

> 对应工作流：**二、叙事架构设计阶段 → 个人线场景表**
> 本文档定义五条个人线（6月下旬\~7月上旬）的完整场景结构、选择点逻辑和剧情走向。
> **设计原则：内容解耦，剧情紧凑，选择多样化，避免重复感。**

***

## 1. 个人线结构总览

```
┌─────────────────────────────────────────────────────────────────┐
│  共通线终点：烟花大会（6月16日）                                    │
│  ├── 选择邀请的角色 → 进入对应个人线                                │
│  └── 未满足任何条件 → Bad End（单身线）                            │
├─────────────────────────────────────────────────────────────────┤
│  个人线时间轴（6月下旬 ~ 7月上旬）                                  │
│  ├── 第一阶段：关系确认与首次约会（6月17日~6月22日）                 │
│  ├── 第二阶段：核心事件（6月23日~6月28日）                          │
│  ├── 第三阶段：感情升温与告白（6月29日~7月5日）                      │
│  └── 结局：7月上旬，暑假前最后一天                                  │
└─────────────────────────────────────────────────────────────────┘
```

> **个人线起始时间**：6月17日（周四），烟花大会后的第四天
> **个人线结束时间**：7月上旬，暑假开始前一天
> **时间设定**：未放暑假，剧情发生在上学期间的周末和放学后

***

## 2. 个人线通用结构

每条个人线包含以下阶段：

| 阶段   | 时间          | 内容        | 说明               |
| ---- | ----------- | --------- | ---------------- |
| 第一阶段 | 6月17日\~22日  | 关系确认与首次约会 | 确认"恋人"或"准恋人"关系   |
| 第二阶段 | 6月23日\~28日  | 核心事件      | 角色专属剧情           |
| 第三阶段 | 6月29日\~7月5日 | 感情升温与告白   | 正式告白，确立关系        |
| 结局   | 7月上旬        | 结局        | 进入个人线后必定进入结局，无分歧 |

> **选项规则**：进入个人线后无关键选项（除约会地点选择外），必定进入结局

## 8. 个人线通用系统

### 8.1 结局规则

| 规则    | 说明             |
| ----- | -------------- |
| 无分歧   | 进入个人线后必定进入结局   |
| 无关键选项 | 除约会地点选择外，无其他选项 |
| 单结局   | 每条个人线只有一个结局    |

### 8.2 约会地点选项

| 选项   | 适用角色 | 说明          |
| ---- | ---- | ----------- |
| 海边   | 全员   | 看海、散步、沙滩上交流 |
| 电影院  | 全员   | 看电影、分享爆米花   |
| 咖啡店  | 全员   | 聊天、吃甜点      |
| 家庭餐厅 | 全员   | 吃饭、聊天       |
| 花店   | 全员   | 买花、拍照       |

### 8.3 个人线与共通线的解耦

- 个人线剧情完全独立，不影响其他角色线
- 个人线内无其他角色出场（除必要剧情外）
- 个人线结局不影响二周目游玩
