# 《君と描くパレット》— 剧情数据库结构

## 1. 核心变量表

### 1.1 路线分支变量

| 变量名 | 类型 | 范围 | 初始值 | 说明 |
|--------|------|------|--------|------|
| `current_chapter` | int | 1-6 | 1 | 当前章节 |
| `current_route` | string | — | "common" | 当前路线（common/ayane/chinatsu/chisa/akane/koharu） |
| `game_day` | int | 1-365 | 1 | 游戏内天数 |
| `play_count` | int | 1+ | 1 | 游玩轮次 |

### 1.2 主角成长变量

| 变量名 | 类型 | 范围 | 初始值 | 说明 |
|--------|------|------|--------|------|
| `yuma_growth` | string | — | "outsider" | 悠真成长阶段（outsider/glimmer/coloring/belonging/awakening/colored） |
| `yuma_courage` | bool | false | 悠真是否在关键选择中做出勇敢选择 |
| `yuma_belonging` | bool | false | 悠真是否找到归属感 |

## 3. 结局解锁标志位

| 标志位 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `ending_ayane_good` | bool | false | 绫音线Good End达成 |
| `ending_ayane_bad` | bool | false | 绫音线Bad End达成 |
| `ending_chinatsu_good` | bool | false | 千夏线Good End达成 |
| `ending_chinatsu_bad` | bool | false | 千夏线Bad End达成 |
| `ending_chisa_good` | bool | false | 千纱线Good End达成 |
| `ending_chisa_bad` | bool | false | 千纱线Bad End达成 |
| `ending_akane_good` | bool | false | 茜线Good End达成 |
| `ending_akane_bad` | bool | false | 茜线Bad End达成 |
| `ending_koharu_good` | bool | false | 小春线Good End达成 |
| `ending_koharu_bad` | bool | false | 小春线Bad End达成 |
| `ending_outsider` | bool | false | 局外人结局达成 |
| `ending_true` | bool | false | True End达成 |

**True End解锁条件**：
```
IF ending_ayane_good AND ending_chinatsu_good AND ending_chisa_good 
   AND ending_akane_good AND ending_koharu_good THEN
    unlock(ending_true)
ENDIF
```

### 4.2 路线分歧逻辑

```
SWITCH fireworks_choice:
    CASE "ayane":   route = "ayane"
    CASE "chinatsu": route = "chinatsu"
    CASE "chisa":   route = "chisa"
    CASE "akane":   route = "akane"
    CASE "koharu":  route = "koharu"
    CASE "alone":   route = "outsider"
END SWITCH
```

## 5. 数据库关系图

```
┌──────────────┐     ┌──────────────┐
│  路线分支变量  │     │  成长变量     │
│  route/flags │     │  yuma_*      │
└──────┬───────┘     └──────┬───────┘
       │                     │
       └──────────┬──────────┘
                  │
           ┌──────▼──────┐
           │  选择点系统  │
           │  choices     │
           └──────┬──────┘
                  │
           ┌──────▼──────┐
           │  剧情节点    │
           │  story_nodes │
           └──────┬──────┘
                  │
           ┌──────▼──────┐
           │  标志位系统  │
           │  flags       │
           └──────┬──────┘
                  │
           ┌──────▼──────┐
           │  结局系统    │
           │  endings     │
           └─────────────┘
```

**数据流向**：选择点 → 标志位变化 → 剧情节点跳转 → 结局判定
