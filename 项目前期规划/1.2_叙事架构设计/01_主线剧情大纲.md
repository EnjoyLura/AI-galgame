## 2. 个人线大纲

### 2.1 白鳥綾音线 ——「被看见不等于被审判」

**青春之壁**：被爱的牢笼束缚——母亲用"为你好"温柔地否定了她的一切，而她连恨的资格都没有

| 层级  | 揭示内容                                                                                                                    | 触发条件        |
| --- | ----------------------------------------------------------------------------------------------------------------------- | ----------- |
| 第一层 | 綾音是优雅高傲的学生会长，对大多数人不假辞色，用命令代替请求，用讽刺代替关心                                                                                  | 共通线自然揭示     |
| 第二层 | 綾音秘密地画漫画，梦想是成为漫画家而非母亲期望的"名门淑女"。她的高傲是铠甲——小时候她把画给最信任的朋友看，那个朋友却告诉了母亲。从此她学会了不暴露弱点                                           | 共通线后期/个人线初期 |
| 第三层 | 綾音的母亲翠子不是暴力地否定她，而是温柔地否定——"画画是很好的爱好，但你是白鳥家的女儿，你的未来不是画纸上的线条。妈妈帮你安排了更好的路。"这种温柔的否定比暴力更可怕，因为它让綾音无法恨母亲，只能恨自己"为什么想要的和妈妈期望的不一样" | 个人线中期       |

### 2.2 星野千夏线 ——「向日葵也需要阳光」

**悠真的成长**：悠真用"怎样都好吧"逃避一切——不靠近、不选择、不负责。但千夏是唯一一个不接受他这种态度的人。她会直接闯进他的世界，拉他出去跑步、逼他说出自己的想法、不给他旁观的机会。千夏让悠真明白：逃避不是保护，旁观不是安全。当悠真看到千夏——那个永远笑着的女孩——终于允许自己在他面前脆弱时，他第一次感受到了"被信任"的重量。两人在一起，是因为悠真学会了"不再逃避"，而千夏学会了"允许自己被照亮"。

### 2.4 篠崎茜线 ——「深夜便利店的温柔」

| Bad End    | 小春选择继续把心意藏在"妹妹"的名字下，继续画着哥哥的背影，继续做那个最了解他却最不能靠近他的人。悠真看着阳台上盛开的铃兰，不知道那朵花在等谁。而小春的便当依然营养均衡，校服依然熨得平整——只是素描本上，背影的画又多了一页。                                                                                                                                                    | —                         |

**悠真的成长**：悠真在重组家庭中觉得自己是"外人"——继父客气、妹妹乖巧，但都是"别人的家人"。但小春让他意识到：他一直在用"局外人"的身份逃避，而小春一直在用"妹妹"的身份忍耐。小春比任何人都了解他——他的逃避、他的恐惧、他的"怎样都好吧"——她全看在眼里，却选择了默默守护而不是闯入。当悠真看到小春蹲在他门外睡着的那一刻，他才明白：这个家里，一直有一个人比他更害怕失去。两人在一起，是因为悠真学会了"打破身份的枷锁"，而小春学会了"理性地放手一搏"。

## 3. True End ——「君色の季節」

**解锁条件**：5条个人线Good End全部通关

### 核心主题

> 悠真在帮助五位少女的过程中，逐渐从"局外人"变成了有归属的人。True End不是选择某一个人，而是悠真在毕业式上回顾这一年的感悟——他终于找到了自己的归属。但True End并非否定恋爱，而是揭示：每一段"在一起"的旅程，本质上都是两个人互相选择的过程。

### True End 走向

| 章节       | 核心事件                                                                                                                                     |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| 毕业式前夜    | 悠真独自来到天台，回忆这一年的每个瞬间                                                                                                                      |
| 回忆串联     | 五段记忆交织——綾音走出温室、千夏允许自己脆弱、千紗的告白、茜伸出的手、小春的铃兰。每段记忆都伴随着悠真自己的变化：从局外人→在意→主动→勇敢→有归属。而贯穿所有记忆的，是小春每天早晨递过来的便当、每天傍晚校门口的等待、每个深夜门缝透进的光——她一直在那里，只是他从未看见 |
| 找到归属     | 悠真终于明白：归属不是被给予的，而是自己选择的。他曾经觉得自己是"外人"，是因为他从未试着去选择——选择靠近、选择留下、选择成为别人的不可替代                                                                  |
| 毕业式      | 悠真在樱花纷飞中走出校门，不再是那个局外人的旁观者。他看着手中五段记忆的碎片，微笑着说："谢谢你们，让我知道——属于哪里，是自己选的。"                                                                     |
| Epilogue | 数年后，悠真成为了一名教师，回到星见丘学园。他站在讲台上，看着台下那些迷茫的面孔，微笑着说："你们一定也能找到自己的归属——只要愿意迈出那一步。"窗外樱花纷飞，和多年前那个春天一模一样                                             |

> **设计说明**：True End聚焦悠真个人的成长与自我救赎，是对"互相救赎"主题的最终回答——帮助别人的过程，本身就是找到归属的过程。True End不与任何角色建立恋爱关系，但它为每条个人线的Good End赋予了更深层的意义：每一段"在一起"都不是偶然，而是两个人互相选择的必然结果。
