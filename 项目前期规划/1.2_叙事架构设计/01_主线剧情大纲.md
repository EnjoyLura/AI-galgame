# 《君色の季節》— 主线剧情大纲

## 1. 共通线剧情大纲

### 第一章：局外人的春天（4月·相遇与重逢篇）

**核心事件**：新学期开始，悠真在重组家庭中感到疏离，与五位少女相遇/重逢

| 场景 | 内容 | 选择点 | 关联角色 |
|------|------|--------|----------|
| 早晨出门 | 悠真和小春一起出门上学，小春递上便当说"哥哥，今天多放了玉子烧"，悠真随口说了声谢谢 | — | 小春 |
| 入学式 | 悠真在樱花纷飞中走进星见丘学园，内心独白："又是一年，什么都不会改变吧。" | — | — |
| 樱花隧道 | 上学路上，一个女生站在樱花隧道中回头——是千紗。七年不见的青梅竹马，重逢了 | 认出她叫名字 / 犹豫不敢认 | 千紗 |
| 教室重逢 | 千紗和悠真分在同一个班，她微笑着说"好久不见"，但眼神里有距离感 | 主动搭话 / 点头示意 | 千紗 |
| 学生会长 | 白鳥綾音在入学式上优雅致辞，但悠真注意到她下台后揉了揉手腕——像是在忍耐什么。她对悠真说"你看起来没什么干劲，这样可不行"——语气是命令，但眼神里有一丝好奇 | 在意 / 不在意 | 綾音 |
| 独自午餐 | 午休时发现千夏一个人在天台吃便当，便当做得很精致——像是在认真对待只有一个人的生活。她笑着说"一个人很自由啊"，但空荡荡的天台只有风在回应 | 好奇搭话 / 各自安静 | 千夏 |
| 图书馆邂逅 | 去图书馆还书，发现茜在角落安静地整理图书——她穿着洗得发白但叠得整整齐齐的旧制服，动作很利落。悠真帮她捡掉落的书时，她小声说了"谢谢"就低下头继续工作 | 帮她捡书 / 悄悄离开 | 茜 |
| 回家晚餐 | 晚上一家人吃饭，小春做了味增汤，继父客气地问"学校怎么样"，悠真回答"还行"后就沉默了。小春安静地给悠真添了一碗汤 | — | 小春/家庭 |
| 小春的画 | 经过小春房间，门半开着，看到她在画画——画的是一个少年的背影，桌上散落着素描本，每一页都是同一个人 | 询问画的是谁 / 默默走开 | 小春 |
| 放学选择 | 第一天放学，自由行动 | **选择去哪里** | 全员 |

## 2. 个人线大纲

### 2.1 白鳥綾音线 ——「被看见不等于被审判」

**青春之壁**：被爱的牢笼束缚——母亲用"为你好"温柔地否定了她的一切，而她连恨的资格都没有

| 层级 | 揭示内容 | 触发条件 |
|------|----------|----------|
| 第一层 | 綾音是优雅高傲的学生会长，对大多数人不假辞色，用命令代替请求，用讽刺代替关心 | 共通线自然揭示 |
| 第二层 | 綾音秘密地画漫画，梦想是成为漫画家而非母亲期望的"名门淑女"。她的高傲是铠甲——小时候她把画给最信任的朋友看，那个朋友却告诉了母亲。从此她学会了不暴露弱点 | 共通线后期/个人线初期 |
| 第三层 | 綾音的母亲翠子不是暴力地否定她，而是温柔地否定——"画画是很好的爱好，但你是白鳥家的女儿，你的未来不是画纸上的线条。妈妈帮你安排了更好的路。"这种温柔的否定比暴力更可怕，因为它让綾音无法恨母亲，只能恨自己"为什么想要的和妈妈期望的不一样" | 个人线中期 |

| 章节 | 核心事件 | 关键选择 |
|------|----------|----------|
| 第四章：画室的门 | 悠真被綾音"命令"帮忙搬学生会资料，意外发现旧校舍的秘密画室。綾音第一反应是冰冷地赶他走——"你不该在这里。忘掉你看到的。"但第二天她又若无其事地叫悠真来帮忙，只是这次……画室的门没有锁 | 推门进去 / 在门外等她 |
| 第五章：温柔的否定 | 翠子发现了画室，没有撕画，而是温柔地把画稿收进纸箱："妈妈帮你保管，你专心准备学生会和文化祭的事。"綾音看着空荡荡的墙壁，没有哭——她的表情比哭更可怕，是完全的空白。她对所有人微笑，比以前更完美，但眼睛里什么都没有了 | 陪她坐在空画室里 / 去找翠子谈话 |
| 第六章：被看见 | 綾音在悠真面前崩溃——不是大哭，而是第一次卸下高傲："我恨她……不，我恨我自己。我连恨她的资格都没有，因为她是为了我好。可是……我画了十年的画，她一次都没有看过。一次都没有。"悠真没有说"坚持梦想"，而是说："你以前画过一幅星空，记得吗？我第一次觉得星星是暖的。"綾音愣住了——他看过她的画，而且记住了 | 说"我一直在看" / 紧紧握住她的手 |
| Good End | 綾音在文化祭上展出了一幅画——画的是旧校舍画室里，一个少年安静地坐在角落喝草莓牛奶，阳光从窗户照进来。标题是《被看见不等于被审判》。翠子在画前站了很久，第一次没有说话。綾音对悠真说："你从来没有告诉我应该怎么做——你只是在那里。但就是'在那里'这件事，让我第一次觉得，被看见不等于被审判。"悠真第一次觉得——他不是局外人，因为有人选择让他留在她的世界里 | — |
| Bad End | 綾音在压力下放弃漫画，成为母亲期望的"完美女儿"。悠真路过旧校舍，画室的门锁着，透过门缝只看到空荡荡的墙壁和一盒过期的草莓牛奶 | — |

**悠真的成长**：悠真在重组家庭中不敢做自己——怕添麻烦、怕被排斥。但綾音让他看到：不表达自己，才是对"家"最大的辜负。綾音走出温室的勇气，也给了悠真走出"局外人"身份的勇气。两人在一起，是因为悠真学会了"做自己"，而綾音学会了"被爱不等于被控制，被看见不等于被审判"。

### 2.2 星野千夏线 ——「向日葵也需要阳光」

**青春之壁**：她照亮所有人，却从不允许自己脆弱——"总是坚强"本身就是她的牢笼

| 层级 | 揭示内容 | 触发条件 |
|------|----------|----------|
| 第一层 | 千夏是班上最开朗的人，总是第一个打招呼、第一个帮忙、第一个笑。她看起来自由自在，独居也过得很好。田径部的学妹们崇拜她，同学们依赖她 | 共通线自然揭示 |
| 第二层 | 千夏的阳光不是面具，但有一个盲区——她照顾所有人的情绪，唯独不允许自己示弱。她的"没事的！"不是在骗人，而是在骗自己——她已经分不清"我没事"和"我不能有事"的区别了 | 共通线后期/个人线初期 |
| 第三层 | 千夏最深的恐惧不是孤独，而是被看见脆弱之后就没有人再需要她了——"如果我不坚强，还有谁会需要我？"她照亮所有人不是因为善良，而是因为害怕：如果她不再发光，就没有理由留在任何人的世界里 | 个人线中期 |

| 章节 | 核心事件 | 关键选择 |
|------|----------|----------|
| 第四章：晨雾中的背影 | 悠真早起路过操场，看到千夏在晨雾中奔跑——那一刻她没有笑，表情是他从未见过的安静。后来悠真撞见千夏在器材室角落蹲着，手在发抖，她笑着说"没事，只是有点喘"。悠真第一次没有相信她的笑——"你不用对我笑。"千夏愣住了。从来没有人对她说过这句话 | 追问她到底怎么了 / 相信她说的话 |
| 第五章：不允许的脆弱 | 千夏发着高烧还在帮同学准备文化祭，悠真发现她一个人在天台靠着墙发抖，嘴里还在说"我没事的，还能撑"。悠真没有说"你可以不用坚强"，只是安静地坐在她旁边——没有走。千夏靠在他肩上说"我好累"——然后自己被这句话吓到了，猛地坐起来想用"没事的！"收回去。但悠真没有给她机会，他只是**没有走** | 强行带她回家休息 / 安静地陪着她 |
| 第六章：向日葵也需要阳光 | 千夏终于崩溃——不是大哭，而是**第一次允许自己不坚强**。她在悠真面前没有笑，声音很小："我一直以为，只要我一直笑、一直坚强，就会有人需要我。但如果我不坚强……还有谁会需要我？"悠真没有说"你不用坚强"，而是说："你跑步的时候不笑，但那个表情比笑的时候更像你。"千夏愣住了——他看过她跑步。他看过她不笑的样子。而且他没有走 | 说"我需要你" / 紧紧握住她的手 |
| Good End | 千夏学会了允许自己脆弱。在操场上，千夏对悠真说："以前我跑步，是为了不让自己停下来想那些不允许想的事情。但现在……我想和你一起跑。不是为了逃避什么——是因为和你在一起的时候，我不用跑也能停下来了。"悠真握住她的手："那就别跑了——走也可以。"千夏笑着哭了："这话说得好肉麻啊……但我好喜欢。"悠真第一次觉得——原来"被需要"不只是被依赖，更是被信任到可以展示脆弱 | — |
| Bad End | 千夏继续做所有人的太阳，不允许自己有阴影。悠真路过她的公寓，窗户亮着灯，但千夏的笑声只在学校里才有。晨雾中她还在跑——因为停下来就会开始想那些不允许想的事情 | — |

**悠真的成长**：悠真用"怎样都好吧"逃避一切——不靠近、不选择、不负责。但千夏是唯一一个不接受他这种态度的人。她会直接闯进他的世界，拉他出去跑步、逼他说出自己的想法、不给他旁观的机会。千夏让悠真明白：逃避不是保护，旁观不是安全。当悠真看到千夏——那个永远笑着的女孩——终于允许自己在他面前脆弱时，他第一次感受到了"被信任"的重量。两人在一起，是因为悠真学会了"不再逃避"，而千夏学会了"允许自己被照亮"。

### 2.3 七瀬千紗线 ——「樱花树下的重逢」（官方路线）

**青春之壁**：时间的距离——阔别七年，彼此都已改变，重逢的陌生与初恋的回忆

| 层级 | 揭示内容 | 触发条件 |
|------|----------|----------|
| 第一层 | 千紗是悠真小时候的青梅竹马，七年后重逢，但两人之间有微妙的距离 | 共通线自然揭示 |
| 第二层 | 千紗因为频繁搬家，每次刚交到朋友就要离开，已经不敢轻易投入感情 | 共通线后期/个人线初期 |
| 第三层 | 千紗一直记得悠真是她第一个喜欢的人，也是她一直回来的理由。但她害怕——如果靠近后发现悠真已经变了，那她最后的寄托也会消失 | 个人线中期 |

| 章节 | 核心事件 | 关键选择 |
|------|----------|----------|
| 第四章：七年的距离 | 千紗和悠真一起去小时候常去的地方，发现一切都变了，只有樱花隧道还是老样子 | 说"我们也没变" / 说"变了也没关系" |
| 第五章：搬家症候群 | 千紗因为害怕再次搬家而不敢和任何人走得太近，包括悠真 | 告诉她"这次我不会让你走" / 等她慢慢打开心 |
| 第六章：樱花树下的告白 | 千紗终于说出七年前没说出口的话："悠真……我从那时候起，就一直喜欢你。" | 接受 / 犹豫 |
| Good End | 悠真握住千紗的手："七年前我没能说出口——我也一直喜欢你。这次，我不会再让你一个人了。"两人在樱花隧道下紧紧相拥，花瓣纷飞如七年前一样。悠真第一次觉得——原来"回来"和"等待"都是有意义的。他不再是无色的局外人，因为有人一直在等他。 | — |
| Bad End | 千紗选择不告白，安静地维持"青梅竹马"的距离。悠真在樱花隧道下独自站了很久，不知道自己失去了什么。 | — |

**悠真的成长**：悠真在重组家庭中觉得自己是"多余的人"——母亲有了新家庭，他算什么呢？但千紗让他明白：有人一直在等他回来，他的存在对某个人来说就是意义。千紗的"回来"给了悠真"被等待"的体验，而悠真的"不走"给了千紗"不再离开"的承诺。两人在一起，是因为悠真学会了"珍惜不放手"，而千紗学会了"勇敢留下来"。

### 2.4 篠崎茜线 ——「深夜便利店的温柔」

**青春之壁**：她习惯了什么都靠自己——独立是她的骄傲，也是她不敢依赖别人的理由

| 层级 | 揭示内容 | 触发条件 |
|------|----------|----------|
| 第一层 | 茜是安静的打工少女，礼貌但有点距离感，在便利店打工时比在学校自在 | 共通线自然揭示 |
| 第二层 | 茜的家庭不富裕，母亲打两份工，她自己也每天打工。她不是不想参加活动，而是打工和课业已经占满了时间。她有点敏感——别人无心的玩笑她会在意很久，但她从不表现出来 | 共通线后期/个人线初期 |
| 第三层 | 茜最深的别扭不是"我不配"，而是"我不想成为别人的负担"。她习惯了什么都靠自己，独立是她的骄傲，但也是她不敢依赖任何人的理由。她害怕的不是被拒绝，而是被接受之后发现自己还不起这份好意 | 个人线中期 |

| 章节 | 核心事件 | 关键选择 |
|------|----------|----------|
| 第四章：便利店的温柔 | 悠真经常在茜打工的便利店买东西，一来二去熟悉了。茜记住了他的购买习惯，会给他留常买的面包。某天深夜悠真路过，看到茜在收银台后面小声念自己写的文章，被发现后她笑着说"在背课文呢"——但悠真注意到稿纸上写的是小说，不是课文 | 说"你念得挺好听的" / 说"下次念给我听" |
| 第五章：打折的布丁 | 悠真开始有意无意地多去便利店，发现茜的生活比想象中更忙碌——放学后打工，回家后写作业，深夜才有时间写文章。文化祭要交班费，茜嘴上说"我交不起也没关系"，但悠真看到她在便利店数零钱数了两遍。悠真想帮她，茜拒绝了——"不用了，我自己可以。真的，谢谢你" | 用"我做多了便当"的理由分给她 / 尊重她的拒绝，但下次再来 |
| 第六章：伸出的手 | 茜的文章在作文比赛中获奖，但颁奖式在工作日，她要打工去不了。悠真问她"你想去吗？"，茜犹豫了很久："……想。但打工不能请假，而且……去了也不知道该穿什么。"悠真说"你想去就去，其他的事想办法"。茜第一次没有立刻说"不用了"——她沉默了很久，然后小声说："……你帮我的话，是因为你想帮？不是因为觉得我可怜？" | 说"因为我想" / 握住她的手 |
| Good End | 茜站在颁奖台上念出自己的文章，题目是《深夜便利店的温柔》——"便利店的灯光总是亮着的，不管多晚，不管外面多黑。它不挑客人，不问来路，只是安静地在那里。我以前觉得，独立就是一个人扛所有的事。但后来我发现，便利店的灯之所以一直亮着，是因为有人一直在换灯泡。"下台后，茜对悠真说："我以前觉得，自己的事自己解决是天经地义的。但偶尔被人照顾的感觉……真的很好。"悠真微笑："那以后，换灯泡的事交给我。"茜笑着轻轻锤了他一下："说什么呢……但，嗯，拜托了。"悠真第一次觉得——接受帮助不是示弱，而是一种信任 | — |
| Bad End | 茜继续一个人扛着所有的事，打工、课业、写文章，什么都自己来。悠真路过便利店，茜在收银台后面安静地站着，口袋里揣着揉皱的获奖通知——她没有去领奖，因为"去了也不知道该说什么"。便利店的灯还亮着，但没有人来换灯泡 | — |

**悠真的成长**：悠真觉得自己是母亲"再婚的包袱"——可有可无的存在。但茜让他看到：独立的人也需要被需要。茜什么都自己来，不是因为不需要别人，而是不知道怎么接受别人的好意。悠真意识到——他一直觉得自己"不被需要"，但茜拒绝帮助的方式让他明白：被需要不是被依赖，而是被信任到可以接受你的好意。两人在一起，是因为悠真学会了"被信任"，而茜学会了"偶尔依赖别人也没关系"。

### 2.5 天羽小春线 ——「铃兰的告白」

**青春之壁**：最了解他的人，最不能靠近他——理性选择了不告白，但理性无法消灭感情

| 层级 | 揭示内容 | 触发条件 |
|------|----------|----------|
| 第一层 | 小春是嫁力超群的义妹，家务全能，对哥哥体贴入微。看起来只是"好妹妹" | 共通线自然揭示 |
| 第二层 | 小春对悠真的了解远超普通兄妹——她知道他的一切习惯、情绪、逃避方式。她画的都是悠真的背影，因为"正面……我会心跳加速"。和悠真独处时会变得坦率，会吐槽、会天然呆，这种反差只有悠真能看到 | 共通线后期/个人线初期 |
| 第三层 | 小春的理性是她最大的牢笼。她比任何人都清楚"不应该喜欢哥哥"——重组家庭的微妙、社会的眼光、可能失去一切的后果。她不是不敢告白，而是理性地选择了不告白。但理性无法消灭感情，只会让感情在沉默中发酵成痛苦。她最害怕的不是被拒绝，而是告白之后连"妹妹"的位置都保不住 | 个人线中期 |

| 章节 | 核心事件 | 关键选择 |
|------|----------|----------|
| 第四章：最了解你的人 | 悠真开始注意到小春对他的了解超乎寻常——他随口说"今天有点累"，第二天便当里就多了补充体力的菜；他说"怎样都好吧"，小春平静地说"哥哥说这句话的时候，其实最在意"。悠真第一次意识到：这个妹妹一直在看着自己 | 说"你怎么什么都知道" / 说"谢谢你一直照顾我" |
| 第五章：理性的牢笼 | 小春在帮助悠真分析其他女生的心意时，不小心说出了只有"喜欢一个人"才会知道的话。悠真追问，小春用"因为我是妹妹啊"挡回去。当晚，悠真半夜醒来，看到门缝透进微光——小春蹲在门外，靠着墙睡着了，手里还握着给悠真准备的温水 | 轻轻把她抱回房间 / 叫醒她让她回房间 |
| 第六章：铃兰的告白 | 小春在阳台给铃兰换盆，悠真走过来。小春低着头说："哥哥……铃兰的花语，你知道吗？是'幸福归来'。我种它的时候在想……只要等下去，幸福一定会回来的。"她终于抬起头，眼眶红了："可是……等了这么久，幸福好像只肯站在门外面。"悠真看着她——围裙上沾着泥土，手指因为常年做家务比同龄人粗糙，但那双琥珀色的眼睛里，是他从未见过的、属于一个女人的坚定 | 说"你不用再等了" / 紧紧握住她的手 |
| Good End | 小春终于鼓起勇气："哥哥……我喜欢你。不是妹妹那种喜欢。是想要一直待在你身边、想要你只看着我的那种喜欢。我知道这很任性……但我不想再理性了。"悠真沉默了很久，然后轻轻抱住她："我也……不想只是你的哥哥。你比任何人都了解我——那你应该知道，我一直在等你说出口。"小春哭着笑了："那……铃兰真的会带来幸福呢。"悠真在她耳边说："我们又不是亲兄妹——但就算我们是，我也不会放手。"两人相拥在铃兰的花香中。悠真第一次觉得——这个家，不再只是"温暖的旅馆"了。因为这里有一个人，把所有的爱都藏在日常的每一个细节里，等了他这么久。 | — |
| Bad End | 小春选择继续把心意藏在"妹妹"的名字下，继续画着哥哥的背影，继续做那个最了解他却最不能靠近他的人。悠真看着阳台上盛开的铃兰，不知道那朵花在等谁。而小春的便当依然营养均衡，校服依然熨得平整——只是素描本上，背影的画又多了一页。 | — |

**悠真的成长**：悠真在重组家庭中觉得自己是"外人"——继父客气、妹妹乖巧，但都是"别人的家人"。但小春让他意识到：他一直在用"局外人"的身份逃避，而小春一直在用"妹妹"的身份忍耐。小春比任何人都了解他——他的逃避、他的恐惧、他的"怎样都好吧"——她全看在眼里，却选择了默默守护而不是闯入。当悠真看到小春蹲在他门外睡着的那一刻，他才明白：这个家里，一直有一个人比他更害怕失去。两人在一起，是因为悠真学会了"打破身份的枷锁"，而小春学会了"理性地放手一搏"。

## 3. True End ——「君色の季節」

**解锁条件**：5条个人线Good End全部通关

### 核心主题

> 悠真在帮助五位少女的过程中，逐渐从"局外人"变成了有归属的人。True End不是选择某一个人，而是悠真在毕业式上回顾这一年的感悟——他终于找到了自己的归属。但True End并非否定恋爱，而是揭示：每一段"在一起"的旅程，本质上都是两个人互相选择的过程。

### True End 走向

| 章节 | 核心事件 |
|------|----------|
| 毕业式前夜 | 悠真独自来到天台，回忆这一年的每个瞬间 |
| 回忆串联 | 五段记忆交织——綾音走出温室、千夏允许自己脆弱、千紗的告白、茜伸出的手、小春的铃兰。每段记忆都伴随着悠真自己的变化：从局外人→在意→主动→勇敢→有归属。而贯穿所有记忆的，是小春每天早晨递过来的便当、每天傍晚校门口的等待、每个深夜门缝透进的光——她一直在那里，只是他从未看见 |
| 找到归属 | 悠真终于明白：归属不是被给予的，而是自己选择的。他曾经觉得自己是"外人"，是因为他从未试着去选择——选择靠近、选择留下、选择成为别人的不可替代 |
| 毕业式 | 悠真在樱花纷飞中走出校门，不再是那个局外人的旁观者。他看着手中五段记忆的碎片，微笑着说："谢谢你们，让我知道——属于哪里，是自己选的。" |
| Epilogue | 数年后，悠真成为了一名教师，回到星见丘学园。他站在讲台上，看着台下那些迷茫的面孔，微笑着说："你们一定也能找到自己的归属——只要愿意迈出那一步。"窗外樱花纷飞，和多年前那个春天一模一样 |

> **设计说明**：True End聚焦悠真个人的成长与自我救赎，是对"互相救赎"主题的最终回答——帮助别人的过程，本身就是找到归属的过程。True End不与任何角色建立恋爱关系，但它为每条个人线的Good End赋予了更深层的意义：每一段"在一起"都不是偶然，而是两个人互相选择的必然结果。
