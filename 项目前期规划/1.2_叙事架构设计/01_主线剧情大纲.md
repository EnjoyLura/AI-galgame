# 《君色の季节》— 主线剧情大纲

## 1. 共通线剧情大纲

### 第一章：局外人的春天（4月·相遇与重逢篇）

**核心主题**：悠真在重组家庭中感到疏离，与五位少女相遇/重逢——每一次相遇都在提醒他，他还没有找到自己站的位置

**时间跨度**：4月1日（入学式前日）→ 4月30日（黄金周前）

#### 序幕：4月1日——"旁观者"

**核心主题**：悠真作为"局外人"的日常——家庭中的疏离、商店街中的格格不入、被拒绝的好意、重逢的距离感

## 2. 个人线大纲

### 2.1 白鸟绫音线 ——“被看见不等于被审判”

**青春之壁**：被爱的牢笼束缚——母亲用"为你好"温柔地否定了她的一切，而她连恨的资格都没有

## 3. True End ——“君色の季节”

**解锁条件**：5条个人线Good End全部通关

### 核心主题

> 悠真在帮助五位少女的过程中，逐渐从"局外人"变成了有归属的人。True End不是选择某一个人，而是悠真在毕业式上回顾这一年的感悟——他终于找到了自己的归属。但True End并非否定恋爱，而是揭示：每一段"在一起"的旅程，本质上都是两个人互相选择的过程。

### True End 走向

| 章节       | 核心事件                                                                                                                                     |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| 毕业式前夜    | 悠真独自来到天台，回忆这一年的每个瞬间                                                                                                                      |
| 回忆串联     | 五段记忆交织——绫音走出温室、千夏允许自己脆弱、千纱的告白、茜伸出的手、小春的铃兰。每段记忆都伴随着悠真自己的变化：从局外人→在意→主动→勇敢→有归属。而贯穿所有记忆的，是小春每天早晨递过来的便当、每天傍晚校门口的等待、每个深夜门缝透进的光——她一直在那里，只是他从未看见 |
| 找到归属     | 悠真终于明白：归属不是被给予的，而是自己选择的。他曾经觉得自己是"外人"，是因为他从未试着去选择——选择靠近、选择留下、选择成为别人的不可替代                                                                  |
| 毕业式      | 悠真在樱花纷飞中走出校门，不再是那个局外人的旁观者。他看着手中五段记忆的碎片，微笑着说："谢谢你们，让我知道——属于哪里，是自己选的。"                                                                     |
| Epilogue | 数年后，悠真成为了一名教师，回到星见丘学园。他站在讲台上，看着台下那些迷茫的面孔，微笑着说："你们一定也能找到自己的归属——只要愿意迈出那一步。"窗外樱花纷飞，和多年前那个春天一模一样                                             |

> **设计说明**：True End聚焦悠真个人的成长与自我救赎，是对"互相救赎"主题的最终回答——帮助别人的过程，本身就是找到归属的过程。True End不与任何角色建立恋爱关系，但它为每条个人线的Good End赋予了更深层的意义：每一段"在一起"都不是偶然，而是两个人互相选择的必然结果。
