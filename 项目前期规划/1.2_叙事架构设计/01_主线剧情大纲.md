# 《君色の季節》— 主线剧情大纲

## 1. 共通线剧情大纲

### 第一章：局外人的春天（4月·相遇与重逢篇）

**核心事件**：新学期开始，悠真在重组家庭中感到疏离，与五位少女相遇/重逢

| 场景 | 内容 | 选择点 | 关联角色 |
|------|------|--------|----------|
| 早晨出门 | 悠真和小春一起出门上学，小春递上便当说"哥哥，今天多放了玉子烧"，悠真随口说了声谢谢 | — | 小春 |
| 入学式 | 悠真在樱花纷飞中走进星见丘学园，内心独白："又是一年，什么都不会改变吧。" | — | — |
| 樱花隧道 | 上学路上，一个女生站在樱花隧道中回头——是千紗。七年不见的青梅竹马，重逢了 | 认出她叫名字 / 犹豫不敢认 | 千紗 |
| 教室重逢 | 千紗和悠真分在同一个班，她微笑着说"好久不见"，但眼神里有距离感 | 主动搭话 / 点头示意 | 千紗 |
| 学生会长 | 白鳥綾音在入学式上优雅致辞，但悠真注意到她下台后揉了揉手腕——像是在忍耐什么 | 在意 / 不在意 | 綾音 |
| 独自午餐 | 午休时发现千夏一个人在天台吃便当，看起来很自在，便当做得很精致——像是在认真对待只有一个人的生活 | 好奇搭话 / 各自安静 | 千夏 |
| 图书馆邂逅 | 去图书馆还书，发现茜在角落安静地整理图书——她穿着洗得发白的旧制服 | 帮她捡书 / 悄悄离开 | 茜 |
| 回家晚餐 | 晚上一家人吃饭，小春做了味增汤，继父客气地问"学校怎么样"，悠真回答"还行"后就沉默了。小春安静地给悠真添了一碗汤 | — | 小春/家庭 |
| 小春的画 | 经过小春房间，门半开着，看到她在画画——画的是一个少年的背影，桌上散落着素描本，每一页都是同一个人 | 询问画的是谁 / 默默走开 | 小春 |
| 放学选择 | 第一天放学，自由行动 | **选择去哪里** | 全员 |

### 第二章：颜色的碎片（5-6月·日常篇）

**核心事件**：日常互动加深，初步了解各角色的表面性格，悠真开始感到"也许我可以不再只是旁观"

| 场景 | 内容 | 选择点 | 关联角色 |
|------|------|--------|----------|
| 学生会帮忙 | 被綾音拉去帮忙处理学生会事务，无意中发现她的书包里藏着画满漫画的笔记本 | 好奇翻看 / 装作没看见 | 綾音 |
| 天台的千夏 | 连续几天午休都看到千夏一个人在天台，她总是笑着说"一个人很自由啊"，但空荡荡的天台只有风在回应 | 主动坐过去 / 保持距离 | 千夏 |
| 千紗的距离 | 千紗虽然和悠真同班，但总是保持着微妙的距离——像在试探什么 | 主动约她放学一起走 / 顺其自然 | 千紗 |
| 茜的便当 | 发现茜的便当永远是白饭配一个腌梅干，但她吃得很安静很认真 | 把自己的菜分给她 / 不忍心看 | 茜 |
| 小春的等待 | 每天放学小春都在校门口等悠真一起回家，风雨无阻。悠真说"你不用每次都等"，小春平静地回答"因为等你的时候，我可以看书" | 和她一起走 / 让她先回去 | 小春 |
| 便当时间 | 天台便当成为日常，不同角色会来天台 | 和谁一起吃 | 全员 |
| 社团参观周 | 学校社团参观周，悠真考虑加入某个社团 | **选择参观哪个社团** | 全员 |

### 第三章：夏日的不安（7月·转折篇）

**核心事件**：期末考试、夏日祭、烟花大会，角色表面下的不安开始浮现

| 场景 | 内容 | 选择点 | 关联角色 |
|------|------|--------|----------|
| 期末学习会 | 在家庭餐厅一起复习，日常甜蜜 | 坐在谁旁边 | 全员 |
| 綾音的崩溃 | 无意中看到綾音在美术室偷偷画漫画时被母亲发现，翠子当场撕毁了画稿 | 上前阻止 / 震惊地旁观 | 綾音 |
| 千夏的空房间 | 去千夏家送忘带的资料，发现偌大的公寓里只有她一个人，冰箱上贴着"汇款已到"的便签，旁边是三年前的家庭合照 | 关心她的生活 / 不想多管 | 千夏 |
| 千紗的日记 | 不小心看到千紗的笔记本，里面写着"回到星见市，但一切都不一样了" | 追问 / 等她自己说 | 千紗 |
| 茜的打工 | 深夜路过便利店，看到茜在收银台后面疲惫地打瞌睡 | 进去陪她 / 默默离开 | 茜 |
| 小春的眼泪 | 打雷的夜晚，小春敲响悠真的房门，浑身发抖，但嘴里说的是"哥哥……我没事"。悠真注意到她手里攥着一条叠好的毛巾——她是来给悠真送毛巾的，只是被雷吓到了 | 让她进来 / 说"回自己房间吧" | 小春 |
| 夏日祭 | 夏日祭约会，与好感最高角色一起逛祭典 | **选择和谁一起** | 全员 |
| 烟花大会 | 烟花大会是共通线最终分歧点 | **选择和谁一起看烟花** | 全员 |

## 2. 个人线大纲

### 2.1 白鳥綾音线 ——「温室中的花」

**青春之壁**：被母亲的爱束缚，无法追求自我

| 层级 | 揭示内容 | 触发条件 |
|------|----------|----------|
| 第一层 | 綾音是完美的学生会长，优雅从容，但偶尔流露出疲惫 | 共通线自然揭示 |
| 第二层 | 綾音秘密地画漫画，梦想是成为漫画家而非母亲期望的"名门淑女" | 共通线后期/个人线初期 |
| 第三层 | 綾音的母亲翠子多次撕毁她的画稿，说"画画不能当饭吃，你是白鳥家的女儿"。綾音爱母亲，但这份爱让她窒息 | 个人线中期 |

| 章节 | 核心事件 | 关键选择 |
|------|----------|----------|
| 第四章：画稿的秘密 | 深入了解綾音的学生会工作，发现她深夜在美术室画漫画 | 鼓励她继续画 / 劝她听母亲的话 |
| 第五章：撕裂的画 | 翠子来学校发现綾音在画漫画，当众撕毁画稿，綾音崩溃 | 站在她这边 / 保持中立 |
| 第六章：走出温室 | 綾音决定在文化祭上展出自己的漫画，面对母亲的反对 | 陪她一起面对 / 让她自己决定 |
| Good End | 綾音的漫画在文化祭上获得好评，同学们的认可让翠子第一次看到女儿真正的笑容。綾音在屋上花园对悠真告白："是你让我知道，被爱不一定要被关在笼子里。"两人在夕阳下牵手。悠真第一次觉得——自己不是局外人，因为有人选择了他。 | — |
| Bad End | 綾音在压力下放弃漫画，成为母亲期望的"完美女儿"。悠真在美术室看到空荡荡的画架。 | — |

**悠真的成长**：悠真在重组家庭中不敢做自己——怕添麻烦、怕被排斥。但綾音让他看到：不表达自己，才是对"家"最大的辜负。綾音走出温室的勇气，也给了悠真走出"局外人"身份的勇气。两人在一起，是因为悠真学会了"做自己"，而綾音学会了"被爱不等于被控制"。

### 2.2 星野千夏线 ——「向日葵也需要阳光」

**青春之壁**：她照亮所有人，却从不允许自己脆弱——"总是坚强"本身就是她的牢笼

| 层级 | 揭示内容 | 触发条件 |
|------|----------|----------|
| 第一层 | 千夏是班上最开朗的人，总是第一个打招呼、第一个帮忙、第一个笑。她看起来自由自在，独居也过得很好 | 共通线自然揭示 |
| 第二层 | 千夏的阳光不是面具，但有一个盲区——她照顾所有人的情绪，唯独不允许自己示弱。她从不诉苦、从不求助、从不让人看到她疲惫的样子 | 共通线后期/个人线初期 |
| 第三层 | 千夏从小就被教导"要坚强，不能让爸妈担心"。父母赴海外后，她把"不麻烦别人"变成了信条。她不是不想依靠别人，而是已经不知道怎么开口说"我需要帮助"。她照亮了所有人，却从不允许自己被照亮 | 个人线中期 |

| 章节 | 核心事件 | 关键选择 |
|------|----------|----------|
| 第四章：不灭的向日葵 | 悠真逐渐发现千夏的另一面——她总是笑着，但从不让人看到她累的时候。某天悠真无意撞见千夏在田径部跑完后独自蹲在器材室角落，她笑着说"没事，只是有点喘"，但手在发抖 | 追问她到底怎么了 / 相信她说的话 |
| 第五章：不允许的脆弱 | 千夏在帮助所有人之后终于撑不住了——她发着高烧还在帮同学准备文化祭，悠真发现她一个人在天台靠着墙发抖，嘴里还在说"我没事，还能撑" | 强行带她去保健室 / 温柔地劝她休息 |
| 第六章：向日葵也需要阳光 | 千夏在海边终于崩溃——不是大哭，而是蹲在地上发抖，声音很小："我……其实一直很害怕一个人……但我不能说，因为说了就输了。"悠真第一次看到她哭 | 说"在我面前，你可以不用坚强" / 紧紧抱住她 |
| Good End | 千夏学会了允许自己脆弱。在海边，千夏对悠真说："你知道吗……向日葵也需要阳光。我一直以为自己是太阳，但原来……我也需要被照亮。"悠真握住她的手："那就让我做你的阳光。"千夏笑着哭了："这话说得好肉麻啊……但我好喜欢。"悠真第一次觉得——原来"被需要"不只是被依赖，更是被信任到可以展示脆弱。 | — |
| Bad End | 千夏继续做所有人的太阳，不允许自己有阴影。悠真路过她的公寓，窗户亮着灯，但千夏的笑声只在学校里才有。 | — |

**悠真的成长**：悠真用"怎样都好吧"逃避一切——不靠近、不选择、不负责。但千夏是唯一一个不接受他这种态度的人。她会直接闯进他的世界，拉他出去跑步、逼他说出自己的想法、不给他旁观的机会。千夏让悠真明白：逃避不是保护，旁观不是安全。当悠真看到千夏——那个永远笑着的女孩——终于允许自己在他面前脆弱时，他第一次感受到了"被信任"的重量。两人在一起，是因为悠真学会了"不再逃避"，而千夏学会了"允许自己被照亮"。

### 2.3 七瀬千紗线 ——「樱花树下的重逢」（官方路线）

**青春之壁**：时间的距离——阔别七年，彼此都已改变，重逢的陌生与初恋的回忆

| 层级 | 揭示内容 | 触发条件 |
|------|----------|----------|
| 第一层 | 千紗是悠真小时候的青梅竹马，七年后重逢，但两人之间有微妙的距离 | 共通线自然揭示 |
| 第二层 | 千紗因为频繁搬家，每次刚交到朋友就要离开，已经不敢轻易投入感情 | 共通线后期/个人线初期 |
| 第三层 | 千紗一直记得悠真是她第一个喜欢的人，也是她一直回来的理由。但她害怕——如果靠近后发现悠真已经变了，那她最后的寄托也会消失 | 个人线中期 |

| 章节 | 核心事件 | 关键选择 |
|------|----------|----------|
| 第四章：七年的距离 | 千紗和悠真一起去小时候常去的地方，发现一切都变了，只有樱花隧道还是老样子 | 说"我们也没变" / 说"变了也没关系" |
| 第五章：搬家症候群 | 千紗因为害怕再次搬家而不敢和任何人走得太近，包括悠真 | 告诉她"这次我不会让你走" / 等她慢慢打开心 |
| 第六章：樱花树下的告白 | 千紗终于说出七年前没说出口的话："悠真……我从那时候起，就一直喜欢你。" | 接受 / 犹豫 |
| Good End | 悠真握住千紗的手："七年前我没能说出口——我也一直喜欢你。这次，我不会再让你一个人了。"两人在樱花隧道下紧紧相拥，花瓣纷飞如七年前一样。悠真第一次觉得——原来"回来"和"等待"都是有意义的。他不再是无色的局外人，因为有人一直在等他。 | — |
| Bad End | 千紗选择不告白，安静地维持"青梅竹马"的距离。悠真在樱花隧道下独自站了很久，不知道自己失去了什么。 | — |

**悠真的成长**：悠真在重组家庭中觉得自己是"多余的人"——母亲有了新家庭，他算什么呢？但千紗让他明白：有人一直在等他回来，他的存在对某个人来说就是意义。千紗的"回来"给了悠真"被等待"的体验，而悠真的"不走"给了千紗"不再离开"的承诺。两人在一起，是因为悠真学会了"珍惜不放手"，而千紗学会了"勇敢留下来"。

### 2.4 篠崎茜线 ——「看不见的花」

**青春之壁**：贫穷让她看不见自己的价值，自卑让她不敢靠近任何人

| 层级 | 揭示内容 | 触发条件 |
|------|----------|----------|
| 第一层 | 茜是安静的打工少女，几乎不参加任何课外活动，总是独来独往 | 共通线自然揭示 |
| 第二层 | 茜的家庭非常贫穷，单亲母亲打两份工，她自己也每天打工到很晚 | 共通线后期/个人线初期 |
| 第三层 | 茜其实成绩优秀、文章写得好，但贫穷让她觉得自己"不配"拥有梦想和友情。她用"我不需要"来回避"我想要但得不到"的痛苦 | 个人线中期 |

| 章节 | 核心事件 | 关键选择 |
|------|----------|----------|
| 第四章：便当与尊严 | 悠真发现茜总是吃最便宜的便当，想帮她但怕伤她自尊 | 用"我做多了"的理由分给她 / 直接说"让我帮你" |
| 第五章：红叶谷 | 茜第一次接受悠真的邀请去红叶谷，但穿着旧鞋磨出了血泡，她笑着说"没关系" | 蹲下来帮她处理伤口 / 说"下次穿好一点的鞋来" |
| 第六章：你的价值 | 茜的文章在作文比赛中获奖，但她因为付不起颁奖式的交通费而想放弃 | 替她出交通费并陪她去 / 鼓励她自己想办法 |
| Good End | 茜站在颁奖台上念出自己的文章，题目是《看不见的花》——"即使没有人看见，花也会开。因为开花不是为了被看见，而是因为那是花该做的事。"下台后，茜对悠真说："是你让我知道……我也有开花的权利。"悠真微笑："你本来就在开花，只是你自己没看到。"两人并肩走出礼堂，阳光照在茜的新鞋上。悠真第一次觉得——每个人都有不可替代的价值，包括他自己。 | — |
| Bad End | 茜放弃领奖，继续在自卑中沉默。悠真在公告栏上看到她的获奖名字，旁边是空缺的领奖人。 | — |

**悠真的成长**：悠真觉得自己是母亲"再婚的包袱"——可有可无的存在。但茜让他看到：价值不是别人给的，而是自己创造的。茜在贫穷中依然发光的坚韧，让悠真意识到——他也不是包袱，他也可以成为别人的光。两人在一起，是因为悠真学会了"看见价值"，而茜学会了"相信自己值得"。

### 2.5 天羽小春线 ——「铃兰的告白」

**青春之壁**：最了解他的人，最不能靠近他——理性选择了不告白，但理性无法消灭感情

| 层级 | 揭示内容 | 触发条件 |
|------|----------|----------|
| 第一层 | 小春是嫁力超群的义妹，家务全能，对哥哥体贴入微。看起来只是"好妹妹" | 共通线自然揭示 |
| 第二层 | 小春对悠真的了解远超普通兄妹——她知道他的一切习惯、情绪、逃避方式。她画的都是悠真的背影，因为"正面……我会心跳加速"。和悠真独处时会变得坦率，会吐槽、会天然呆，这种反差只有悠真能看到 | 共通线后期/个人线初期 |
| 第三层 | 小春的理性是她最大的牢笼。她比任何人都清楚"不应该喜欢哥哥"——重组家庭的微妙、社会的眼光、可能失去一切的后果。她不是不敢告白，而是理性地选择了不告白。但理性无法消灭感情，只会让感情在沉默中发酵成痛苦。她最害怕的不是被拒绝，而是告白之后连"妹妹"的位置都保不住 | 个人线中期 |

| 章节 | 核心事件 | 关键选择 |
|------|----------|----------|
| 第四章：最了解你的人 | 悠真开始注意到小春对他的了解超乎寻常——他随口说"今天有点累"，第二天便当里就多了补充体力的菜；他说"怎样都好吧"，小春平静地说"哥哥说这句话的时候，其实最在意"。悠真第一次意识到：这个妹妹一直在看着自己 | 说"你怎么什么都知道" / 说"谢谢你一直照顾我" |
| 第五章：理性的牢笼 | 小春在帮助悠真分析其他女生的心意时，不小心说出了只有"喜欢一个人"才会知道的话。悠真追问，小春用"因为我是妹妹啊"挡回去。当晚，悠真半夜醒来，看到门缝透进微光——小春蹲在门外，靠着墙睡着了，手里还握着给悠真准备的温水 | 轻轻把她抱回房间 / 叫醒她让她回房间 |
| 第六章：铃兰的告白 | 小春在阳台给铃兰换盆，悠真走过来。小春低着头说："哥哥……铃兰的花语，你知道吗？是'幸福归来'。我种它的时候在想……只要等下去，幸福一定会回来的。"她终于抬起头，眼眶红了："可是……等了这么久，幸福好像只肯站在门外面。"悠真看着她——围裙上沾着泥土，手指因为常年做家务比同龄人粗糙，但那双琥珀色的眼睛里，是他从未见过的、属于一个女人的坚定 | 说"你不用再等了" / 紧紧握住她的手 |
| Good End | 小春终于鼓起勇气："哥哥……我喜欢你。不是妹妹那种喜欢。是想要一直待在你身边、想要你只看着我的那种喜欢。我知道这很任性……但我不想再理性了。"悠真沉默了很久，然后轻轻抱住她："我也……不想只是你的哥哥。你比任何人都了解我——那你应该知道，我一直在等你说出口。"小春哭着笑了："那……铃兰真的会带来幸福呢。"悠真在她耳边说："我们又不是亲兄妹——但就算我们是，我也不会放手。"两人相拥在铃兰的花香中。悠真第一次觉得——这个家，不再只是"温暖的旅馆"了。因为这里有一个人，把所有的爱都藏在日常的每一个细节里，等了他这么久。 | — |
| Bad End | 小春选择继续把心意藏在"妹妹"的名字下，继续画着哥哥的背影，继续做那个最了解他却最不能靠近他的人。悠真看着阳台上盛开的铃兰，不知道那朵花在等谁。而小春的便当依然营养均衡，校服依然熨得平整——只是素描本上，背影的画又多了一页。 | — |

**悠真的成长**：悠真在重组家庭中觉得自己是"外人"——继父客气、妹妹乖巧，但都是"别人的家人"。但小春让他意识到：他一直在用"局外人"的身份逃避，而小春一直在用"妹妹"的身份忍耐。小春比任何人都了解他——他的逃避、他的恐惧、他的"怎样都好吧"——她全看在眼里，却选择了默默守护而不是闯入。当悠真看到小春蹲在他门外睡着的那一刻，他才明白：这个家里，一直有一个人比他更害怕失去。两人在一起，是因为悠真学会了"打破身份的枷锁"，而小春学会了"理性地放手一搏"。

## 3. True End ——「君色の季節」

**解锁条件**：5条个人线Good End全部通关

### 核心主题

> 悠真在帮助五位少女的过程中，逐渐从"局外人"变成了有归属的人。True End不是选择某一个人，而是悠真在毕业式上回顾这一年的感悟——他终于找到了自己的归属。但True End并非否定恋爱，而是揭示：每一段"在一起"的旅程，本质上都是两个人互相选择的过程。

### True End 走向

| 章节 | 核心事件 |
|------|----------|
| 毕业式前夜 | 悠真独自来到天台，回忆这一年的每个瞬间 |
| 回忆串联 | 五段记忆交织——綾音走出温室、千夏允许自己脆弱、千紗的告白、茜的绽放、小春的铃兰。每段记忆都伴随着悠真自己的变化：从局外人→在意→主动→勇敢→有归属。而贯穿所有记忆的，是小春每天早晨递过来的便当、每天傍晚校门口的等待、每个深夜门缝透进的光——她一直在那里，只是他从未看见 |
| 找到归属 | 悠真终于明白：归属不是被给予的，而是自己选择的。他曾经觉得自己是"外人"，是因为他从未试着去选择——选择靠近、选择留下、选择成为别人的不可替代 |
| 毕业式 | 悠真在樱花纷飞中走出校门，不再是那个局外人的旁观者。他看着手中五段记忆的碎片，微笑着说："谢谢你们，让我知道——属于哪里，是自己选的。" |
| Epilogue | 数年后，悠真成为了一名教师，回到星见丘学园。他站在讲台上，看着台下那些迷茫的面孔，微笑着说："你们一定也能找到自己的归属——只要愿意迈出那一步。"窗外樱花纷飞，和多年前那个春天一模一样 |

> **设计说明**：True End聚焦悠真个人的成长与自我救赎，是对"互相救赎"主题的最终回答——帮助别人的过程，本身就是找到归属的过程。True End不与任何角色建立恋爱关系，但它为每条个人线的Good End赋予了更深层的意义：每一段"在一起"都不是偶然，而是两个人互相选择的必然结果。
