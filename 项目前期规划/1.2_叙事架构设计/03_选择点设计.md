# 《君色の季節》— 选择点设计

## 1. 选择系统总览

### 1.1 选择点分类

| 类型 | 说明 | 影响 | 出现频率 |
|------|------|------|----------|
| 日常选择 | 放学后去哪里、和谁互动 | 好感度±1~3 | 每日1次 |
| 对话选择 | 对话中的回应方式 | 好感度±2~5 | 每场景0-2次 |
| 关键选择 | 剧情分歧点 | 路线走向、结局 | 每章1-2次 |
| 隐藏选择 | 特殊条件触发 | 依赖度±5~10 | 视条件而定 |

### 1.2 好感度与依赖度

每个选择同时影响好感度和依赖度，两者共同决定结局走向。好感度反映角色对悠真的好感程度，依赖度反映角色对悠真的情感依赖程度。高好感+高依赖 = Good End。

> 好感度/依赖度的具体区间划分和阈值设计见 → [好感度系统](../../角色设计阶段/2.1_角色企划/03_好感度系统.md)

## 4. 隐藏选择系统

### 4.1 触发条件

隐藏选择不在正常对话选项中出现，需要满足特定条件：

| 隐藏选择 | 触发条件 | 内容 | 影响 |
|----------|----------|------|------|
| 綾音的画 | 连续3次选择綾音 + 好感度≥50 | 悠真主动提出看綾音的漫画 | 依赖度+10 |
| 千夏的钥匙 | 千夏好感度≥40 + 去过千夏家2次 | 悠真发现千夏家门没锁，千夏笑着说"因为说不定你会来啊" | 依赖度+8 |
| 千紗的回忆 | 千紗好感度≥50 + 去过樱花隧道3次 | 触发小时候的回忆CG | 依赖度+10 |
| 茜的文章 | 茜好感度≥40 + 在图书馆遇到茜3次 | 悠真发现茜写的文章 | 依赖度+8 |
| 小春的日记 | 小春好感度≥60 + 打雷夜选择让她进来 | 悠真无意看到小春素描本的某一页——画的不是背影，而是悠真笑着看她的正面 | 依赖度+10 |

### 4.2 连续事件链

连续选择同一角色会触发特殊事件链：

| 连续次数 | 触发事件 | 好感度 | 依赖度 |
|----------|----------|--------|--------|
| 连续3次 | 角色主动找悠真搭话 | +5 | +3 |
| 连续5次 | 角色邀请悠真一起放学 | +8 | +5 |
| 连续7次 | 角色向悠真吐露秘密 | +10 | +8 |
| 连续10次 | 角色专属事件触发 | +15 | +10 |

## 5. 选择点设计原则

| 原则 | 说明 |
|------|------|
| 无明显正确答案 | 选项不应让玩家一眼看出哪个是"好"选择 |
| 情感驱动 | 选择应基于"你想怎么回应"而非"哪个分数高" |
| 后果延迟 | 选择的影响不立即显现，而是在后续剧情中逐步体现 |
| 多维度影响 | 同一选择同时影响好感度和依赖度，但方向可能不同 |
| 角色一致性 | 选择应符合角色的性格和当前关系阶段 |
| 避免惩罚感 | Bad End不是惩罚，而是"不够勇敢"的自然结果 |
