## 3. 路线进入机制

### 3.1 好感度累积

共通线中通过日常选择累积好感度，**烟花大会（7月）** 是最终分歧点：

| 条件 | 结果 |
|------|------|
| 单一角色好感远超其他（≥20点差距） | 直接进入该角色个人线 |
| 两个角色好感接近（差距<10点） | 进入好感更高的角色线，但加入另一角色的伏笔 |
| 所有角色好感都很低 | 进入"局外人结局"——悠真独自度过这一年 |

### 3.2 推荐游玩顺序

| 游玩顺序 | 推荐路线 | 理由 |
|----------|----------|------|
| 第1轮 | 七瀬千紗线 | 官方主线，青梅竹马重逢最易入门，情感线最自然 |
| 第2轮 | 篠崎茜线 | 温暖治愈，日常互动多，甜度与深度并存 |
| 第3轮 | 星野千夏线 | 阳光与脆弱的反差，情感层次丰富，与男主互补最强 |
| 第4轮 | 白鳥綾音线 | 家庭控制主题，情感冲击力强 |
| 第5轮 | 天羽小春线 | 禁忌张力最强，嫁力义妹+理性暗恋的反差，需要前四轮的情感铺垫 |
| 第6轮 | True End | 全部线索汇聚，悠真个人成长 |

## 5. 各路线结局定义

### 5.1 Good End 条件

| 路线 | Good End触发条件 |
|------|-----------------|
| 白鳥綾音线 | 第六章选择"陪她一起面对" + 好感度≥75 + 依赖度≥60 |
| 星野千夏线 | 第六章选择"说'在我面前，你可以不用坚强'" + 好感度≥75 + 依赖度≥60 |
| 七瀬千紗线 | 第六章选择"接受" + 好感度≥75 + 依赖度≥60 |
| 篠崎茜线 | 第六章选择"替她出交通费并陪她去" + 好感度≥75 + 依赖度≥60 |
| 天羽小春线 | 第六章选择"说'你不用再等了'" + 好感度≥75 + 依赖度≥60 |

### 5.1.1 Good End 内容摘要

| 路线 | Good End 概要 | 两人在一起的关键 |
|------|--------------|-----------------|
| 白鳥綾音线 | 綾音的漫画在文化祭获奖，母亲终于认可。綾音在屋上花园告白，两人在夕阳下牵手 | 悠真学会了"做自己"，綾音学会了"被爱≠被控制" |
| 星野千夏线 | 千夏不再做所有人的太阳，在海边对悠真说"向日葵也需要阳光"。两人笑着哭了 | 悠真学会了"不再逃避"，千夏学会了"允许自己被照亮" |
| 七瀬千紗线 | 千紗在樱花隧道下告白，悠真回应"我也一直喜欢你"。两人相拥 | 悠真学会了"珍惜不放手"，千紗学会了"勇敢留下来" |
| 篠崎茜线 | 茜站在颁奖台上念出《看不见的花》，下台后和悠真并肩走出礼堂 | 悠真学会了"看见价值"，茜学会了"相信自己值得" |
| 天羽小春线 | 小春鼓起勇气告白，悠真回应"不想只是你的哥哥"。两人相拥在铃兰花香中 | 悠真学会了"打破身份枷锁"，小春学会了"理性地放手一搏" |

### 5.2 Bad End 条件

| 路线 | Bad End触发条件 |
|------|----------------|
| 白鳥綾音线 | 选择"让她自己决定" 或 好感度不足 |
| 星野千夏线 | 选择"紧紧抱住她"（不够深入，只是保护而非理解） 或 好感度不足 |
| 七瀬千紗线 | 选择"犹豫" 或 好感度不足 |
| 篠崎茜线 | 选择"鼓励她自己想办法"（不够主动） 或 好感度不足 |
| 天羽小春线 | 选择"紧紧握住她的手"（只是保护而非回应） 或 好感度不足 |

### 5.2.1 Bad End 内容摘要

| 路线 | Bad End 概要 | 遗憾的核心 |
|------|-------------|-----------|
| 白鳥綾音线 | 綾音放弃漫画，成为"完美女儿"。悠真在美术室看到空荡荡的画架 | 悠真在关键时刻选择了旁观 |
| 星野千夏线 | 千夏继续做所有人的太阳，不允许自己有阴影。悠真路过她的公寓，窗户亮着灯，但千夏的笑声只在学校里才有 | 悠真只是保护了她，没有让她学会脆弱 |
| 七瀬千紗线 | 千紗不告白，维持"青梅竹马"的距离。悠真在樱花隧道下独自站了很久 | 悠真在告白面前犹豫，错过了七年的等待 |
| 篠崎茜线 | 茜放弃领奖，继续在自卑中沉默。公告栏上她的名字旁是空缺的领奖人 | 悠真没有主动伸出手，茜继续觉得自己"不配" |
| 天羽小春线 | 小春把心意藏在"妹妹"的名字下，继续做最了解他却最不能靠近他的人。素描本上背影的画又多了一页 | 悠真只是握住了她的手，没有回应她的等待 |

### 5.3 Bad End 设计原则

- 每个Bad End都是"差一点就能到达"的遗憾
- Bad End不惩罚玩家，而是展示"如果不够勇敢会怎样"
- Bad End有独立的CG和结局曲
- Bad End的核心情感是"遗憾"而非"绝望"
- 每个Bad End都暗示了通往Good End的方向

### 5.4 局外人结局（共通线Bad End）

- 触发条件：烟花大会时所有角色好感度都很低
- 内容：悠真独自在河边看烟花，感叹"又是旁观的一年"
- 作用：让玩家意识到"不主动靠近任何人"的后果
- 特殊：此结局会揭示悠真在重组家庭中的完整内心独白

## 6. 千紗线作为官方主线的特殊设计

作为官方女主路线，千紗线在以下方面有特殊处理：

| 方面 | 特殊设计 |
|------|----------|
| 共通线权重 | 千紗在共通线中的出场频率和剧情比重最高 |
| 情感铺垫 | 悠真对千紗的初恋回忆贯穿共通线始终 |
| 路线入口 | 千紗线的进入门槛最低（初始好感加成最高） |
| True End关联 | 千紗线的主题"珍惜与归属"是True End的核心基调 |
| 宣传定位 | 所有宣传素材以千紗为第一女主形象 |
