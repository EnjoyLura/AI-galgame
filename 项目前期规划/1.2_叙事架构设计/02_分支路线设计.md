## 3. 路线进入机制

### 3.1 固定选项分歧

共通线中通过固定选项决定路线走向，**烟花大会（7月）** 是最终分歧点：

| 烟花大会选择 | 进入路线 |
|-------------|----------|
| 和绫音一起看烟花 | 白鸟绫音线 |
| 和千夏一起看烟花 | 星野千夏线 |
| 和千纱一起看烟花 | 七瀬千纱线 |
| 和茜一起看烟花 | 筿崎茜线 |
| 和小春一起看烟花 | 天羽小春线 |
| 独自看烟花 | 局外人结局 |

### 3.2 共通线铺垫

虽然路线由烟花大会最终决定，但共通线中的选择会影响：
- 对应角色的铺垫剧情是否出现（选择关注某角色→解锁该角色的深层场景）
- 进入个人线后的对话细节变化（共通线中与该角色的互动会被回忆引用）
- 悠真对该角色的认知深度（影响个人线初期的内心独白内容）

### 3.3 推荐游玩顺序

| 游玩顺序 | 推荐路线 | 理由 |
|----------|----------|------|
| 第1轮 | 七瀬千纱线 | 官方主线，青梅竹马重逢最易入门，情感线最自然 |
| 第2轮 | 筿崎茜线 | 温暖治愈，日常互动多，甜度与深度并存 |
| 第3轮 | 星野千夏线 | 阳光与脆弱的反差，情感层次丰富，与男主互补最强 |
| 第4轮 | 白鸟绫音线 | 家庭控制主题，情感冲击力强 |
| 第5轮 | 天羽小春线 | 禁忌张力最强，嫁力义妹+理性暗恋的反差，需要前四轮的情感铺垫 |
| 第6轮 | True End | 全部线索汇聚，悠真个人成长 |

## 5. 各路线结局定义

### 5.1 Good End 条件

| 路线 | Good End触发条件 |
|------|-----------------|
| 白鸟绫音线 | 第六章选择"陪她一起面对" |
| 星野千夏线 | 第六章选择"说'在我面前，你可以不用坚强'" |
| 七瀬千纱线 | 第六章选择"接受" |
| 筿崎茜线 | 第六章选择"陪她去颁奖式" |
| 天羽小春线 | 第六章选择"说'你不用再等了'" |

### 5.1.1 Good End 内容摘要

| 路线 | Good End 概要 | 两人在一起的关键 |
|------|--------------|-----------------|
| 白鸟绫音线 | 绫音的漫画在文化祭获奖，母亲终于认可。绫音在屋上花园告白，两人在夕阳下牵手 | 悠真学会了"做自己"，绫音学会了"被爱≠被控制" |
| 星野千夏线 | 千夏不再做所有人的太阳，在海边对悠真说"向日葵也需要阳光"。两人笑着哭了 | 悠真学会了"不再逃避"，千夏学会了"允许自己被照亮" |
| 七瀬千纱线 | 千纱在樱花隧道下告白，悠真回应"我也一直喜欢你"。两人相拥 | 悠真学会了"珍惜不放手"，千纱学会了"勇敢留下来" |
| 筿崎茜线 | 茜站在颁奖台上念出《深夜便利店的温柔》，下台后对悠真说"偶尔被人照顾的感觉真的很好" | 悠真学会了"被信任"，茜学会了"偶尔依赖别人也没关系" |
| 天羽小春线 | 小春鼓起勇气告白，悠真回应"不想只是你的哥哥"。两人相拥在铃兰花香中 | 悠真学会了"打破身份枷锁"，小春学会了"理性地放手一搏" |

### 5.2 Bad End 条件

| 路线 | Bad End触发条件 |
|------|----------------|
| 白鸟绫音线 | 选择"让她自己决定" |
| 星野千夏线 | 选择"紧紧抱住她"（不够深入，只是保护而非理解） |
| 七瀬千纱线 | 选择"犹豫" |
| 筿崎茜线 | 选择"鼓励她自己决定"（不够主动） |
| 天羽小春线 | 选择"紧紧握住她的手"（只是保护而非回应） |

### 5.2.1 Bad End 内容摘要

| 路线 | Bad End 概要 | 遗憾的核心 |
|------|-------------|-----------|
| 白鸟绫音线 | 绫音放弃漫画，成为"完美女儿"。悠真在美术室看到空荡荡的画架 | 悠真在关键时刻选择了旁观 |
| 星野千夏线 | 千夏继续做所有人的太阳，不允许自己有阴影。悠真路过她的公寓，窗户亮着灯，但千夏的笑声只在学校里才有 | 悠真只是保护了她，没有让她学会脆弱 |
| 七瀬千纱线 | 千纱不告白，维持"青梅竹马"的距离。悠真在樱花隧道下独自站了很久 | 悠真在告白面前犹豫，错过了七年的等待 |
| 筿崎茜线 | 茜继续一个人扛着所有的事，口袋里揣着揉皱的获奖通知——她没有去领奖。便利店的灯还亮着，但没有人来换灯泡 | 悠真没有伸出手，茜继续觉得"我自己可以" |
| 天羽小春线 | 小春把心意藏在"妹妹"的名字下，继续做最了解他却最不能靠近他的人。素描本上背影的画又多了一页 | 悠真只是握住了她的手，没有回应她的等待 |

### 5.3 Bad End 设计原则

- 每个Bad End都是"差一点就能到达"的遗憾
- Bad End不惩罚玩家，而是展示"如果不够勇敢会怎样"
- Bad End有独立的CG和结局曲
- Bad End的核心情感是"遗憾"而非"绝望"
- 每个Bad End都暗示了通往Good End的方向

### 5.4 局外人结局（共通线Bad End）

- 触发条件：烟花大会选择"独自看烟花"
- 内容：悠真独自在河边看烟花，感叹"又是旁观的一年"
- 作用：让玩家意识到"不主动靠近任何人"的后果
- 特殊：此结局会揭示悠真在重组家庭中的完整内心独白

## 6. 千纱线作为官方主线的特殊设计

作为官方女主路线，千纱线在以下方面有特殊处理：

| 方面 | 特殊设计 |
|------|----------|
| 共通线权重 | 千纱在共通线中的出场频率和剧情比重最高 |
| 情感铺垫 | 悠真对千纱的初恋回忆贯穿共通线始终 |
| 路线入口 | 千纱线的进入门槛最低（青梅竹马重逢的情感最自然） |
| True End关联 | 千纱线的主题"珍惜与归属"是True End的核心基调 |
| 宣传定位 | 所有宣传素材以千纱为第一女主形象 |
