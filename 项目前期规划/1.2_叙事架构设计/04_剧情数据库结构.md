# 《君色の季節》— 剧情数据库结构

## 1. 核心变量表

### 1.1 角色好感度变量

| 变量名 | 类型 | 范围 | 初始值 | 说明 |
|--------|------|------|--------|------|
| `affection_ayane` | int | 0-100 | 0 | 白鳥綾音好感度 |
| `affection_chinatsu` | int | 0-100 | 0 | 星野千夏好感度 |
| `affection_chisa` | int | 0-100 | 5 | 七瀬千紗好感度（初始加成：青梅竹马） |
| `affection_akane` | int | 0-100 | 0 | 篠崎茜好感度 |
| `affection_koharu` | int | 0-100 | 3 | 天羽小春好感度（初始加成：义妹） |

### 1.2 角色依赖度变量

| 变量名 | 类型 | 范围 | 初始值 | 说明 |
|--------|------|------|--------|------|
| `dependency_ayane` | int | 0-100 | 0 | 白鳥綾音依赖度 |
| `dependency_chinatsu` | int | 0-100 | 0 | 星野千夏依赖度 |

| `dependency_akane` | int | 0-100 | 0 | 篠崎茜依赖度 |
| `dependency_koharu` | int | 0-100 | 5 | 天羽小春依赖度（初始加成：义妹） |

### 1.3 主角成长变量

| 变量名 | 类型 | 范围 | 初始值 | 说明 |
|--------|------|------|--------|------|
| `yuma_growth` | int | 0-100 | 0 | 悠真成长度 |

| 变量名 | 类型 | 范围 | 初始值 | 说明 |

## 3. 结局解锁标志位

| 标志位 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `ending_ayane_good` | bool | false | 綾音线Good End达成 |
| `ending_ayane_bad` | bool | false | 綾音线Bad End达成 |
| `ending_chinatsu_good` | bool | false | 千夏线Good End达成 |
| `ending_chinatsu_bad` | bool | false | 千夏线Bad End达成 |
| `ending_chisa_good` | bool | false | 千紗线Good End达成 |
| `ending_chisa_bad` | bool | false | 千紗线Bad End达成 |
| `ending_akane_good` | bool | false | 茜线Good End达成 |
| `ending_akane_bad` | bool | false | 茜线Bad End达成 |
| `ending_koharu_good` | bool | false | 小春线Good End达成 |
| `ending_koharu_bad` | bool | false | 小春线Bad End达成 |
| `ending_outsider` | bool | false | 局外人结局达成 |
| `ending_true` | bool | false | True End达成 |

**True End解锁条件**：
```
IF ending_ayane_good AND ending_chinatsu_good AND ending_chisa_good 
   AND ending_akane_good AND ending_koharu_good THEN
    unlock(ending_true)
ENDIF
```

### 4.2 路线分歧逻辑

```
IF affection_ayane >= MAX(all) + 20 THEN route = "ayane"
ELSE IF affection_chinatsu >= MAX(all) + 20 THEN route = "chinatsu"
ELSE IF affection_chisa >= MAX(all) + 20 THEN route = "chisa"
ELSE IF affection_akane >= MAX(all) + 20 THEN route = "akane"
ELSE IF affection_koharu >= MAX(all) + 20 THEN route = "koharu"
ELSE IF MAX(all_affections) < 30 THEN route = "outsider"
ELSE route = highest_affection
```

## 5. 数据库关系图

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  好感度变量   │     │  依赖度变量   │     │  成长变量     │
│ affection_*  │     │ dependency_* │     │  yuma_*      │
└──────┬───────┘     └──────┬───────┘     └──────┬───────┘
       │                     │                     │
       └──────────┬──────────┘                     │
                  │                                │
           ┌──────▼──────┐                         │
           │  选择点系统  │                         │
           │  choices     │                         │
           └──────┬──────┘                         │
                  │                                │
           ┌──────▼──────┐                         │
           │  剧情节点    │◄────────────────────────┘
           │  story_nodes │
           └──────┬──────┘
                  │
           ┌──────▼──────┐
           │  标志位系统  │
           │  flags       │
           └──────┬──────┘
                  │
           ┌──────▼──────┐
           │  结局系统    │
           │  endings     │
           └─────────────┘
```

**数据流向**：选择点 → 变量变化 → 标志位触发 → 剧情节点跳转 → 结局判定
