## 2. 核心概念（Elevator Pitch）

> 重组家庭的男孩桐生悠真，在"家"中始终找不到自己的位置——母亲再婚后，他有了继父和妹妹，表面上一切和谐，但他觉得自己像个局外人。升入星见丘学园二年级的春天，他依然用旁观者的目光看着青春流逝。然而，五位各有心结的少女闯入了他的生活：被母亲的爱束缚的温室之花、照亮所有人却不允许自己脆弱的向日葵少女、阔别多年重逢的初恋青梅竹马、在贫穷中低头的灰姑娘、以及最了解他却最不能靠近他的嫁力义妹。当他选择靠近她们、了解她们的内心、帮助她们面对困境时，他也在被她们改变着——尤其是那个唯一不接受他"怎样都好吧"态度的向日葵少女，硬是闯进了他的世界，不给他旁观的机会。**每一条路线的终点，都是两个人的互相救赎——他帮她跨过青春之壁，她帮他走出"无色"的虚无，最终在一起。**

***

## 3. 核心卖点

| 卖点          | 说明                                                                  |
| ----------- | ------------------------------------------------------------------- |
| **美少女角色群**  | 5位性格鲜明、立绘精美的女主角，覆盖主流萌属性（被束缚的大小姐/阳光向日葵少女/重逢初恋/自卑灰姑娘/嫁力义妹），每位都有三层内心深度 |
| **青春恋爱日常**  | 大量温馨有趣的日常互动——放学后的便当、周末的约会、文化祭的准备、夏日的烟花大会，营造沉浸式青春体验                  |
| **互相救赎叙事**  | 不是单向拯救，而是双向治愈——男主帮助女主解决问题，女主也帮助男主找到归属，每条路线都是两个人的成长                  |
| **禁忌与纯爱并存** | 妹妹线提供禁忌张力，青梅竹马线提供纯爱共鸣，每条路线都有独特的情感冲击                                 |
| **真实青春共鸣**  | 每个角色的烦恼都来自真实的青春困境——过度保护、不允许脆弱、距离感、贫穷、禁忌之恋，引发玩家强烈共鸣                  |

## 5. 题材定位分析

### 5.1 题材组合矩阵

| 维度    | 选择               | 理由                      |
| ----- | ---------------- | ----------------------- |
| 时代背景  | 现代日本高中           | Galgame经典舞台，玩家熟悉度高，代入感强 |
| 超自然要素 | 无                | 纯现实题材，聚焦真实青春情感          |
| 情感基调  | 温馨治愈为主，偶有酸涩与禁忌张力 | 日常甜蜜+情感深度+禁忌刺激          |
| 叙事结构  | 共通线→个人线          | 经典Galgame结构，每条路线聚焦一个角色  |
| 核心冲突  | 青春困境vs成长+禁忌vs纯爱  | 每个人都面对自己的"青春之壁"         |

### 5.2 竞品差异化

| 竞品          | 我们的差异                   |
| ----------- | ----------------------- |
| 《CLANNAD》   | 加入重组家庭+禁忌妹妹线，情感维度更复杂    |
| 《白色相簿2》     | 无修罗场，每条路线独立，情感基调更温暖     |
| 《想要传达给你的爱恋》 | 青梅竹马重逢+初恋设定，更有情感冲击力     |
| 《缘之空》       | 妹妹线更注重情感铺垫和伦理挣扎，而非肉体关系  |
| 国产Galgame   | 日系校园+重组家庭+禁忌元素，填补国内市场空白 |

### 5.3 目标受众画像

| 维度   | 描述                        |
| ---- | ------------------------- |
| 年龄   | 16-30岁                    |
| 性别   | 男性为主                      |
| 游戏经验 | 有视觉小说/Galgame游玩经验，喜爱纯爱系作品 |
| 消费习惯 | 愿意为高质量恋爱体验付费50-100元       |
| 平台偏好 | Steam为主                   |
| 情感需求 | 渴望被理解、被治愈的温暖体验，对禁忌恋爱情有独钟  |
| 社区参与 | B站、贴吧、NGA、微博Galgame社区     |
