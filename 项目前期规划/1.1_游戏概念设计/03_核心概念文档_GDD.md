# 《君色の季節》— 核心概念文档 (GDD)

## 1. 游戏核心机制

### 1.1 好感度系统

- 每个可攻略角色独立好感度，通过日常选择、约会事件、关键对话累积
- 好感度影响：日常对话变化、额外事件触发、结局走向
- 隐藏参数"依赖度"：反映角色对主角的情感依赖程度，影响后期关键分支和结局质量
- 好感度+依赖度共同决定结局类型（友情/恋爱/Good End）

> 好感度与依赖度的具体参数区间、阈值设计见 → [好感度系统](../../角色设计阶段/2.1_角色企划/03_好感度系统.md)

### 1.2 日常选择系统

- 每天放学后的自由行动时间，选择去哪里/和谁互动
- 不同选择触发不同日常事件
- 连续选择同一角色会触发"连续事件链"，关系加速发展
- 选择独自行动会触发主角内心独白，推进自我认知

### 1.3 季节事件系统

- 每个季节有固定的季节事件（夏日祭、文化祭、圣诞节等）
- 季节事件中与好感最高角色触发专属剧情
- 季节事件是不可跳过的关键节点
- 错过的日常事件不会重演（无时间回溯机制）

### 1.4 "青春之壁"系统

- 每个角色都有一个核心心理困境，称为"青春之壁"
- 青春之壁分为三层，随好感度和剧情推进逐层揭示
- 突破第三层需要特定条件（好感度+关键选择+季节事件）
- 突破青春之壁是进入Good End的必要条件

### 1.5 "悠真成长"系统

悠真作为男主，同样有自己的"青春之壁"——重组家庭中的局外人感。他的成长与每条路线的女主紧密绑定，体现"互相救赎"的核心。

| 成长阶段 | 悠真变化 | 对应台词变化 |
|----------|----------|-------------|
| 局外人 | 旁观者心态，在家庭中感到疏离 | "怎样都好吧。" |
| 微光 | 开始在意某个人的事 | "她……好像在勉强自己。" |
| 着色 | 主动想要帮助对方 | "我想为她做点什么。" |
| 归属 | 感受到被需要的重量 | "原来被需要是这种感觉。" |
| 觉醒 | 做出勇敢的选择，不再逃避 | "这次，我不会再退缩了。" |
| 有色 | 找到自己的归属和意义 | "我不再是局外人了。" |

> 成长阶段的具体触发阈值、成长度变量设计见 → [好感度系统](../../角色设计阶段/2.1_角色企划/03_好感度系统.md)

**各路线中悠真的成长主题**：

| 路线 | 悠真学到的 |
|------|-----------|
| 綾音线 | 追求自我的勇气 |
| 千夏线 | 允许自己被照亮 |
| 千紗线 | 珍惜不放手 |
| 茜线 | 看见别人看不见的价值 |
| 小春线 | 打破理性与身份的枷锁 |

## 2. 美术风格定义

| 维度 | 方向 |
|------|------|
| 整体风格 | 日系二次元，明亮清新的校园画风，参考《想要传达给你的爱恋》/《千恋万花》 |
| 色调倾向 | 整体明亮温暖；春-粉绿、夏-蓝白、秋-橙红、冬-蓝紫 |
| 立绘风格 | 高精度立绘，注重表情细腻度和服装质感，每角色12种以上表情差分 |
| 背景风格 | 日系校园写实背景，注重光影和季节感，天空表现力强 |
| CG风格 | 恋爱氛围感构图，注重角色表情和肢体语言，关键告白场景全屏CG |
| UI风格 | 清新简约风，主色调为天蓝+白色+淡粉，季节元素随游戏进度变化 |

> 角色色彩设计（代表色、色彩意象、服装倾向）见 → [角色概念设定](../../角色设计阶段/2.1_角色企划/01_角色概念设定.md)
> 关键CG场景规划（13张CG清单与场景描述）见 → [CG清单](../../美术资源生产阶段/3.2_CG插画/01_CG清单.md)
