# 《君と描くパレット》— 核心概念文档 (GDD)

## 1. 游戏核心机制

### 1.1 路线分支系统

- 不设好感度数值，通过**固定选项**直接决定剧情走向
- 共通线中每个关键选择点明确指向不同路线，玩家选择"和谁在一起"即走向对应角色
- **烟花大会**是共通线最终分歧点，此选择锁定进入哪条个人线
- 个人线中每个选择直接决定剧情分支，不累积数值，选项本身即结果
- 路线分支以**flowchart**形式可视化展示，玩家可在流程图中回顾已解锁/未解锁的剧情节点

> 路线分支的详细flowchart和选择点设计见 → [选择点设计](../../项目前期规划/1.2_叙事架构设计/03_选择点设计.md)

### 1.2 日常选择系统

- 每天放学后的自由行动时间，选择去哪里/和谁互动
- 不同选择触发不同日常事件，同时决定后续可用的剧情分支
- 选择不是"累积好感"，而是"打开/关闭"对应的剧情路径
- 选择独自行动会触发主角内心独白，推进自我认知

### 1.3 关键事件系统

- 故事中有固定的关键事件（夏日祭、文化祭、圣诞节等）
- 关键事件中与当前路线角色触发专属剧情
- 关键事件是不可跳过的关键节点

### 1.4 "青春之壁"系统

- 每个角色都有一个核心心理困境，称为"青春之壁"
- 青春之壁分为三层，随剧情推进逐层揭示
- 突破第三层需要在关键选择点做出正确选择
- 突破青春之壁是进入Good End的必要条件

### 1.5 "悠真成长"系统

悠真作为男主，同样有自己的"青春之壁"——重组家庭中的局外人感。他的成长与每条路线的女主紧密绑定，体现"互相救赎"的核心。

| 成长阶段 | 悠真变化 | 对应台词变化 |
|----------|----------|-------------|
| 局外人 | 旁观者心态，在家庭中感到疏离 | "怎样都好吧。" |
| 微光 | 开始在意某个人的事 | "她……好像在勉强自己。" |
| 着色 | 主动想要帮助对方 | "我想为她做点什么。" |
| 归属 | 感受到被需要的重量 | "原来被需要是这种感觉。" |
| 觉醒 | 做出勇敢的选择，不再逃避 | "这次，我不会再退缩了。" |
| 有色 | 找到自己的归属和意义 | "我不再是局外人了。" |

> 成长阶段的具体触发条件见 → [选择点设计](../../项目前期规划/1.2_叙事架构设计/03_选择点设计.md)

**各路线中悠真的成长主题**：

| 路线 | 悠真学到的 |
|------|-----------|
| 绫音线 | 追求自我的勇气 |
| 千夏线 | 允许自己被照亮 |
| 千纱线 | 珍惜不放手 |
| 茜线 | 被信任的重量 |
| 小春线 | 打破理性与身份的枷锁 |

## 2. 美术风格定义

| 维度 | 方向 |
|------|------|
| 整体风格 | 日系二次元，明亮清新的校园画风，参考《想要传达给你的爱恋》/《千恋万花》 |
| 色调倾向 | 整体明亮温暖，以天蓝、白色、淡粉为主调 |
| 立绘风格 | 高精度立绘，注重表情细腻度和服装质感，每角色12种以上表情差分 |
| 背景风格 | 日系校园写实背景，注重光影和氛围感，天空表现力强 |
| CG风格 | 恋爱氛围感构图，注重角色表情和肢体语言，关键告白场景全屏CG |
| UI风格 | 清新简约风，主色调为天蓝+白色+淡粉 |

> 角色色彩设计（代表色、色彩意象、服装倾向）见 → [角色概念设定](../../角色设计阶段/2.1_角色企划/01_角色概念设定.md)
