## 1. 男主：桐生悠真

| 项目 | 内容 |
|------|------|
| 身份 | 星见丘学园二年级，重组家庭的长子 |
| 家庭 | 母亲离婚后带他改嫁，继父带了一个女儿（天羽小春） |
| 性格 | 温和但疏离，习惯旁观，不主动靠近任何人 |
| 困境 | 在"家"中找不到归属——不是不被爱，而是不知道自己该站在哪个位置 |
| 核心台词 | "怎样都好吧。" |
| 社团 | 未加入（故事开始时） |

### 1.1 悠真的内心困境

悠真的问题来自重组家庭中的"局外人感"——不是被虐待或被忽视，而是那种微妙的、说不出口的疏离。

| 困境层次 | 表现 | 内心独白 |
|----------|------|----------|
| 局外感 | 在家里总觉得自己是"桐生家的"，而继父和妹妹是"天羽家的" | "我好像住在一个很温暖的旅馆里。" |
| 无归属 | 不知道自己该站在哪个位置——不是亲生儿子，也不是完全的外人 | "妈有了新的家庭，我算什么呢？" |
| 逃避亲近 | 对继父客气疏离，对妹妹保持"哥哥"的距离，不敢真正靠近任何人 | "太近了会受伤吧——反正怎样都好。" |
| 无方向 | 没有梦想、没有目标，觉得未来怎样都无所谓 | "连自己属于哪里都不知道，谈什么将来。" |
| 自我否定 | 觉得自己的存在是母亲"再婚的包袱" | "如果妈没有带着我，她大概会更幸福吧。" |

**核心矛盾**：悠真渴望归属、渴望被真正需要，但重组家庭的微妙距离让他不敢靠近任何人。他不是不想要，而是害怕靠近之后发现自己依然是局外人。这种恐惧让他选择了旁观——只要不靠近，就不会受伤。

**救赎契机**：当悠真开始帮助五位少女面对她们的问题时，他第一次感受到了"被需要"的重量。而当他发现自己真的可以成为某个人"不可替代的存在"时，他才明白——归属不是被给予的，而是自己选择的。

### 1.2 各路线中悠真的成长主题

| 路线 | 悠真学到的 | 对应悠真的困境突破 |
|------|-----------|-------------------|
| 绫音线 | 追求自我的勇气 | 从"在别人家里不敢做自己"→"有勇气选择自己想要的" |
| 千夏线 | 允许自己被照亮 | 从"害怕靠近会受伤"→"不靠近才是最大的伤害" |
| 千纱线 | 珍惜不放手 | 从"怎样都好吧"→"不想失去就要说出来" |
| 茜线 | 看见别人看不见的价值 | 从"觉得自己是包袱"→"每个人都有不可替代的价值" |
| 小春线 | 打破理性与身份的枷锁 | 从"被身份定义"→"选择自己想成为的人" |

## 2. 白鸟绫音

| 项目 | 内容 |
|------|------|
| 姓名 | 白鸟绫音（しらとり あやね） |
| 年龄 | 16岁 / 高二 |
| 身高 | 162cm |
| 血型 | A型 |
| 生日 | 11月3日 |
| 萌属性 | 傲娇/成绩优秀的高傲大小姐/被束缚的温室花/小恶魔 |
| 身份 | 星见丘学园二年级，学生会会长，年级第一常客 |
| 成绩 | 年级前三，各科均衡优秀，唯独美术课故意不拿高分——"画得再好也不能当饭吃"是母亲的口头禅 |
| 家庭 | 家庭美满，母亲宠爱过度且严厉，反对她的爱好——认为画画浪费时间影响学习 |
| 代表色 | 藤紫色 |
| 社团 | 学生会（会长） |
| 青春之壁 | 被爱的牢笼束缚——母亲用"为你好"否定了她的一切，连她唯一的梦想都被定义为"浪费时间"，而她连恨的资格都没有 |

### 2.1 外貌

一头光泽的浅紫色长发，通常用深紫色的缎带扎成低马尾。五官精致端正，表情总是从容不迫——但仔细看会发现她的从容里有一层薄冰，像是不允许自己出现任何裂缝。眼睛是深紫色，看人的时候带着一种天然的距离感，不是冷漠，而是"不轻易让人靠近"的自觉。身材纤细，穿校服时每一个细节都一丝不苟——领带的角度、裙摆的长度、甚至袜子的褶皱都恰到好处。但偶尔，她的手指会无意识地摩挲手腕——那是长时间握笔画草稿留下的茧，她总是用袖口遮住。

### 2.2 性格

**表面：优雅高傲的"冰之女王"**

绫音不是对谁都温柔的大小姐。她对大多数人不假辞色，说话辛辣但不出恶言，有自己的一套标准——达不到的人她懒得搭理，做对了她会认真认可。她用"命令"代替"请求"，用"评价"代替"关心"，用"讽刺"代替"玩笑"。但她的高傲不是恶意——她做错了会低头认错，对值得尊敬的人会表示敬意，只是说话的方式永远带着那层不容侵犯的壳。

在学业上，绫音是无可挑剔的优等生。年级前三的成绩是她和母亲之间心照不宣的契约——"只要成绩够好，妈妈就不会说太多。"她把成绩当作维持平衡的筹码，每一次满分都是对母亲隐形压力的回应。但讽刺的是，她唯一真正热爱的东西——画画——却从不在成绩单上。美术课的分数她故意维持在中等偏上，因为如果拿了满分，母亲会冷笑着说"画画满分能有什么用？有这个时间不如多刷两套数学题"。

**内面：傲娇又害怕被看见的少女**

绫音的高傲是铠甲，而铠甲之下藏着一个典型的傲娇——嘴上说着"不需要"但眼神会追着看，说着"不关我事"但什么都在意。小时候她曾经把画给最信任的人看，那个人却把这件事告诉了母亲。从此她学会了**不暴露弱点**——"如果没有人看到真实的你，就没有人能否定真实的你。"她的完美不是天性，而是防御机制。她不是不想被人理解，而是已经不知道怎么在不设防的情况下与人相处。

她的傲娇体现在每一个细微处：悠真生病请假时她会"恰好"路过他家门口，"恰好"带着学生会资料，"恰好"问他身体怎么样了——然后补一句"只是作为学生会长确认考勤，别多想"。她会在悠真和其他女生说话时皱眉，但被问到时只会说"我皱眉和你有什么关系"。她不是不坦率，而是坦率这件事本身对她来说就是一种弱点——一旦承认在意，就等于给了别人伤害自己的武器。

**对悠真的特殊态度：教科书级傲娇+小恶魔**

绫音对悠真会流露出只有他能看到的一面——用优雅的命令指使他做事，享受他的窘态，偶尔露出恶作剧得逞的微笑。她会故意用敬语和他说话看他手足无措，会在他面前优雅地评价其他女生然后观察他的反应，会说"你今天的表情很有趣"来掩饰自己在偷看他。

但这一切的真实动机只有一个：想确认他在不在，想知道他有没有在看自己。她不会说"我今天想见你"，而是说"学生会需要人手"；不会说"谢谢你来"，而是说"效率还行"；不会说"明天也来"，而是说"还有资料没整理完"。她用命令包裹邀请，用评价代替关心，用讽刺藏起真心——教科书级别的傲娇。但做错了她会认真道歉——"刚才的话过分了。对不起。"这种"做错就认"的坦率，是她最可爱的地方。

**反差萌细节**：
- 完全不会做饭，连泡面都能煮糊。试过给悠真做便当，结果是灾难——但她把失败品藏在身后不让他看，耳根通红地说"才、才没有做给你！只是顺便多出来的！"
- 食量小且偏食，对食物没什么兴趣，但悠真给她买的草莓牛奶她会不动声色地喝完——然后第二天说"昨天那种草莓牛奶……味道不算差。你以后买的时候顺便再多带一盒，只是顺便。"
- 被悠真无意中夸奖时会条件反射地说"那是当然的"，然后别过脸——耳尖通红。如果悠真继续盯着她看，她会恼羞成怒："看什么看！"
- 方向感极差，去没去过的地方一定会迷路，但绝不承认——"我没有迷路，只是在探索最优路线。"如果悠真说"我带你走吧"，她会先瞪他一眼然后默默跟在他身后
- 考试拿了年级第一，被老师表扬时面无表情地说"正常发挥"。但回到学生会办公室后，会偷偷把成绩单摆在桌上最显眼的位置——"只是忘了收起来，不是想让你看"
- 看到悠真的考试成绩后会皱眉——"你这种成绩还想毕业吗。"第二天书桌上会出现一份绫音手写的重点笔记，字迹工整到不可思议，但最后一页用橡皮擦过的地方隐约能看到铅笔写的小字："加油"

### 2.3 日常行为

**秘密画室**：绫音放学后会去旧校舍的一个废弃教室——她把它改造成了私人画室，墙上贴满了漫画草稿和角色设定。这是她的"城堡"，没有人被允许进入。画室的门永远锁着，钥匙她随身携带，连书包都不放。

**指使悠真**：绫音会用学生会事务的名义把悠真叫来——"帮我搬资料""这个文件你整理一下""陪我去检查社团活动"。看起来是命令，其实是在确认"他会不会来"。每一次悠真出现在学生会办公室门口，她都会在心里松一口气——但他不会知道。

**草莓牛奶**：午休时绫音会独自在学生会办公室处理事务，桌上永远放着一盒草莓牛奶。没有人知道那是她自己买的还是谁给的——但悠真注意到，只有他路过便利店顺手带了一盒放在她桌上的时候，她才会喝第二盒。问她为什么，她头也不抬："补充糖分有助于提高工作效率。——不是因为你买的。"

### 2.4 与悠真的关系

**初期：命令与试探**。绫音用学生会会长的身份接近悠真，用"命令"来掩饰"在意"。她不是在追求悠真，而是在试探——试探他能不能承受她的高傲，试探他会不会像其他人一样被她的刺扎走。每一次悠真没有走，她的防线就松动一点。

**中期：留一条缝的门**。当悠真意外发现画室后，绫音第一反应是冰冷地赶他走。但第二天，她又若无其事地叫他来帮忙——只是这次，画室的门没有锁。她不会主动邀请他进入，但她会**留一条缝**，等他自己推门。这是绫音式的温柔——不是敞开怀抱，而是"如果你想来，门没有关死"。

**后期：第一次被看见**。当绫音在悠真面前崩溃时，她不是大哭，而是**第一次卸下高傲**——"我连恨她的资格都没有。"悠真没有说"坚持梦想"，也没有说"和妈妈谈谈"，他只是说："你画画的时候，表情和平时不一样。"这句话让绫音意识到——他看过她的画，而且**记住了**。不是评价好不好，而是看到了画背后的她。

### 2.5 三层内心深度

| 层级 | 揭示内容 | 触发条件 |
|------|----------|----------|
| 第一层 | 绫音是优雅高傲的学生会长，成绩年级前三，对大多数人不假辞色，但偶尔流露出疲惫。她用命令代替请求，用讽刺代替关心 | 共通线自然揭示 |
| 第二层 | 绫音秘密地画漫画，梦想是成为漫画家而非母亲期望的"名门淑女"。她的高傲是铠甲——小时候她把画给最信任的朋友看，那个朋友却告诉了母亲。从此她学会了不暴露弱点。她在学业上的出色表现是维持平衡的筹码——"只要成绩够好，妈妈就不会说太多" | 共通线后期/个人线初期 |
| 第三层 | 绫音的母亲翠子不是暴力地否定她，而是**温柔地否定**——"画画是很好的爱好，但你觉得这有用吗？你每次画画的时间如果用来学习，年级第一肯定是你的。妈妈不是不让你画，只是不想看你把时间浪费在没有价值的事情上。"这种温柔的否定比暴力更可怕，因为它把绫音唯一的热爱定义为"浪费时间"，让她无法恨母亲，只能恨自己"为什么想要的和妈妈期望的不一样"。更残酷的是，她用成绩换来的平静是脆弱的——一旦成绩下滑，母亲就会归咎于画画，"你看，我就说画画会影响学习" | 个人线中期 |

### 2.6 核心矛盾

绫音的核心矛盾是**"被爱"与"被看见"的对立**——以及**傲娇外壳下"想被理解"与"不敢暴露真心"的拉扯**。

母亲翠子爱她，但这份爱是一个牢笼——翠子用"为你好"温柔地否定了绫音的一切真实渴望。翠子没有撕画，没有骂她，只是轻轻把画推回来，微笑着说"画画是很好的爱好，但你觉得这有用吗？你花三个小时画一幅画的时间，用来做一套数学真题，分数就能前进五名。妈妈不是不让你画——只是不想看你浪费时间。"这种温柔的否定让绫音无法反抗——因为反抗就意味着否定母亲的爱，而她知道母亲是真的爱她。但这份爱让她窒息，因为它**只看见了"白鸟家的女儿"和"成绩单上的分数"，看不见"绫音"**。

更令人窒息的是成绩捆绑：绫音用年级前三的成绩维持着微妙的和平。每一次考试都是一场交易——交出足够漂亮的成绩单，换取母亲对画画睁一只眼闭一只眼。但这个交易让她活在持续的焦虑中：一旦某次考试失误，母亲就会精准地归咎于画画——"你看，我就说画画会影响学习吧。"成绩不是她骄傲的资本，而是她赎回梦想的人质。

绫音的高傲和傲娇是这层窒息的产物——如果没有人看到真实的她，就没有人能否定真实的她。她用完美和距离保护自己，用"不需要"来掩盖"想要"，用"不关我事"来藏起"很在意"。但代价是永远孤独。她不是不想被人理解，而是已经不知道怎么在不设防的情况下与人相处。

### 2.7 绫音的"崩溃"

在个人线中，当翠子把绫音画室的画稿全部收走时，绫音没有哭——她的表情比哭更可怕，是完全的空白。她对所有人微笑，比以前更完美，成绩反而上升了——因为画画的时间被彻底剥夺，她只能用刷题来填补那片空白。但眼睛里的光消失了。

她的崩溃是在悠真面前触发的，触发点不是一个大的事件，而是一个微小的瞬间：悠真在学生会办公室里，不经意地递给她一张便签——上面是他随手画的、笨拙的动物涂鸦。绫音盯着那张涂鸦看了很久，然后手开始发抖。

"我恨她……不，我恨我自己。我连恨她的资格都没有，因为她是为了我好。她说画画是浪费时间——我花十年时间做一件'浪费时间'的事，我是不是很可笑？可是……我画了十年的画，她一次都没有看过。一次都没有。在她眼里，我做的事情只是'影响学习的坏习惯'……"

这句话是绫音最核心的痛——不是画被拿走了，而是**十年了，母亲从来没有看过她画的是什么**。母亲否定的不是画，而是绫音这个人本身。更痛的是，她无法用成绩来反驳——因为她的成绩确实好，而正因为成绩好，母亲才更不解："你成绩这么好，为什么要把时间浪费在画画上？"

## 3. 星野千夏

### 3.1 外貌

一头利落的深蓝色短发，发尾微微翘起——不是精心打理的，而是跑步时被风吹出来的形状。眼睛是明亮的琥珀色，看人的时候总是带着笑意，像阳光照在湖面上。身材修长结实，小腿上有常年跑步留下的肌肉线条，但手臂很细——她总是穿长袖，因为怕别人看到她跑步摔倒留下的疤。笑起来的时候整张脸都在发光，是那种让人忍不住也跟着笑的类型。但仔细看会发现，她的笑容从来没有"中间状态"——要么灿烂到刺眼，要么完全没有。她不会苦笑，不会勉强地笑，因为对她来说，笑只有"开"和"关"两个状态。

穿校服时永远把袖子卷到手肘，领带系得松松垮垮——不是不在意，而是随时准备跑步。书包里永远塞着跑鞋，便当盒是最大号的那种——一个人要做两个人的分量，因为"吃得好才能跑得快"。手上戴着一块旧款运动手表，表带已经磨得发白——那是父母出国前送的，她从来没换过。

### 3.2 性格

**表面：照亮所有人的"向日葵"**

千夏是班上最开朗的人——第一个打招呼、第一个帮忙、第一个笑。她对谁都热情，但不是那种没心没肺的热闹，而是一种让人安心的温暖。她会注意到谁今天心情不好，会主动帮人值日，会在别人尴尬时自然地解围。田径部的学妹们崇拜她，同学们依赖她，她看起来自由自在，独居也过得很好。她的口头禅是"没事的！"——无论什么情况，她总是先说这三个字。

**内面：不允许自己有阴影的少女**

千夏的阳光不是面具，她是真的开朗。但她的开朗有一个盲区——她把所有人的情绪都照顾到了，唯独不允许自己脆弱。她从不诉苦、从不求助、从不让人看到她疲惫的样子。这不是在"装坚强"，而是她已经把"坚强"内化成了本能——就像呼吸一样，她不记得自己是什么时候开始不允许自己示弱的。

从小每次父母不在身边，她都告诉自己"要坚强，不能让爸妈担心"。久而久之，"坚强"变成了她的本能，"不麻烦别人"变成了她的信条。她不是不想依靠别人，而是已经不知道怎么开口说"我需要帮助"。她照亮了所有人，却从不允许自己被照亮。

**对悠真的特殊态度：闯入者**

千夏对悠真的方式和其他人完全不同——她不试探、不等待、不给距离，而是直接闯进来。她会拉他去跑步、逼他说出想法、不给他"怎样都好吧"的机会。她不是在追求悠真，而是本能地觉得"这个人不能就这样下去"。但只有悠真注意到——千夏在照顾所有人的时候，从来没有人在照顾她。

**反差萌细节**：
- 做饭非常好吃，便当永远是最大号的，但自己吃的时候总是最后才吃——"先把大家的份准备好"
- 一个人在家的时候会自言自语，因为"太安静了会有回声，有点可怕"
- 害怕打雷但绝不承认——打雷的时候会突然变得特别忙碌，"我只是刚好想整理一下柜子"
- 跑步的时候会笑，但不是开心的笑，而是"只有跑步的时候不用对任何人笑"的释然

### 3.3 日常行为

**晨跑**：千夏每天早上5点起来跑步，风雨无阻。这不是自律，而是"如果停下来，就会开始想那些不允许想的事情"。跑步是她唯一"只为自己"的时间——不用照顾任何人，不用对任何人笑，只有风和自己的呼吸。悠真第一次早起路过操场，看到千夏在晨雾中奔跑的背影——那一刻她没有笑，表情是他从未见过的安静。

**天台午餐**：午休时千夏总是一个人在天台吃便当。便当做得很精致，像是在认真对待只有一个人的生活。她笑着说"一个人很自由啊"，但空荡荡的天台只有风在回应。偶尔有人来天台，她会立刻切换成"开朗模式"——动作之快，像按了开关。

**独居的日常**：千夏的公寓干净整洁，但有一种"样板间"的感觉——太整齐了，像是不允许自己留下生活的痕迹。冰箱上贴着"汇款已到"的便签，旁边是三年前的家庭合照。厨房台面上永远放着两人份的餐具——她做两人份的饭，吃两人份的量，因为"一个人做饭不好控制分量"。但悠真注意到，餐桌的两把椅子，有一把从来没有拉出来过。

### 3.4 与悠真的关系

**初期：闯入与拉扯**。千夏是唯一一个不接受悠真"怎样都好吧"态度的人。她会直接闯进他的世界，拉他出去跑步，逼他说出自己的想法。她不是在追求悠真，而是本能地觉得"这个人不能就这样下去"。但悠真慢慢发现——千夏在照顾所有人的时候，从来没有人在照顾她。她拉着他跑，但自己从来没有被人拉住过。

**中期：第一次没有被笑过去**。某天悠真撞见千夏在器材室角落蹲着，手在发抖，她笑着说"没事，只是有点喘"。悠真第一次没有相信她的笑——"你不用对我笑。"千夏愣住了。从来没有人对她说过这句话。从来没有人觉得她的笑需要被质疑。她不知道该用什么表情面对一个"不允许她笑"的人——于是她第一次在悠真面前没有笑。

**后期：允许自己被照亮**。千夏的崩溃不是面具碎裂，而是**第一次允许自己不坚强**。她不是在悠真面前"暴露了真实的自己"，而是在悠真面前"第一次尝试做不坚强的自己"。她靠在他肩上说"我好累"——然后自己被这句话吓到了，因为这是她十六年来第一次对别人说"我累"。

### 3.5 三层内心深度

| 层级 | 揭示内容 | 触发条件 |

### 3.6 核心矛盾

千夏的核心矛盾是**"被需要"与"被看见"的对立**。

她照亮所有人，不是因为天性如此，而是因为"被需要"是她唯一确认自己存在价值的方式。如果她不坚强，还有谁会需要她？如果她不发光，还有谁会让她留在他们的世界里？她的阳光不是面具，但阳光本身成了她的牢笼——向日葵永远朝着太阳，但没有人想过，向日葵自己也需要被照亮。

她的"不麻烦别人"不是独立，而是恐惧——恐惧一旦开口说"我需要帮助"，别人就会发现她没有自己看起来那么强大，然后像父母一样离开。她不是不想依靠别人，而是已经不知道怎么开口。她把"坚强"当成了存在的理由，把"被需要"当成了被爱的替代品。

### 3.7 千夏的"不笑"

在个人线中，千夏的崩溃不是大哭，不是面具碎裂，而是**第一次在悠真面前没有笑**。

她靠在他肩上，说"我好累"——然后自己被这句话吓到了。她猛地坐起来，下意识地想笑，想用"没事的！"把刚才那句话收回去。但悠真没有给她机会——他只是安静地坐在那里，没有说"你可以不用坚强"，也没有说"我会陪着你"，他只是**没有走**。

千夏看着他没有走的背影，突然明白了一件事：原来"被看见脆弱"不等于"被抛弃"。原来有人可以看见她不笑的样子，然后还留在原地。

她的眼泪不是悲伤，而是**释然**——十六年来第一次，有人看见了她没有笑的样子，然后没有离开。

### 4.1 外貌

一头清秀的黑色长发，通常用樱粉色的发带扎成半扎发。脸上总是挂着温柔的微笑，但仔细看会发现那笑容里有一丝小心翼翼——像是在确认"这样笑对不对"。眼睛是深琥珀色，看着悠真的时候会微微发亮，但一旦被注意到就会立刻移开视线。身材匀称偏纤细，穿校服时会将领结系得格外整齐——这是她在每个新学校养成的习惯，"至少在第一天给人留下好印象"。

### 4.2 性格

**表面：温柔得体、善解人意的"好女孩"**

在外人面前，千纱是那种让人挑不出毛病的女孩——礼貌、体贴、成绩优秀、文艺部安静写东西的文学少女。她能自然地融入任何新环境，因为七年的辗转搬家让她学会了"快速适应"的技能。她会在第一天就记住所有同学的名字，会在别人尴尬时帮忙解围，会在集体活动中默默做最辛苦的部分。但这种"得体"不是天性，而是反复练习出来的生存本能。

- 偶尔会冒出小时候的口癖——"悠真くん"，一旦说出口就会立刻改口"桐生くん"，然后内心疯狂后悔

### 4.3 日常行为

**文艺部活动**：千纱在文艺部安静地写东西，但没有人知道她写的是什么。其实她写的是"旅行笔记"——记录在每一个城市见到的风景、遇到的人、吃到的食物。每一本笔记的最后一页，都写着同一句话："如果有一天回到星见市，我想带悠真くん去看这些地方。"七年的搬家生活让她变成了一个沉默的观察者，而文字是她唯一不会失去的东西。

**放学后**：千纱会找各种理由接近悠真——"文艺部要采访学生会的活动""我写了篇短文想让你看看""这家店的限定甜品只有今天有"。她不是在追求，而是在试探——试探悠真是否还像从前一样，试探自己是否还有资格靠近。

**记录习惯**：千纱有一个随身携带的小相机，拍的不是风景，而是日常的碎片——教室窗台上的阳光、走廊尽头的夕阳、操场上奔跑的背影。她害怕遗忘，所以用照片和文字把一切都留住。但她的相机里，出现最多的背影，始终是同一个人。

### 4.4 与悠真的关系

**小时候**：千纱和悠真是邻居，从小一起长大。悠真是她第一个朋友，也是她第一个喜欢的人。小时候的千纱比现在坦率得多——会拉着悠真的手跑过樱花隧道，会在他耳边说"我们永远在一起吧"，会在打雷的夜晚理直气壮地敲他的门说"我害怕"。那时候的她，不知道"永远"是一个多么奢侈的词。

**分别的七年**：小学四年级，千纱因为父亲工作调动离开了星见市。此后七年辗转了四个城市，每次刚交到朋友就要搬家。她学会了不投入太多感情，学会了笑着说"没关系，我已经习惯了"，学会了在离开时不回头——因为回头就会哭。但她始终没有学会忘记悠真。在每一个新城市的第一个夜晚，她都会翻出小时候的合影，然后在本子上写下"今天又想起你了"。

**重逢**：高二春天，千纱回到了星见市。她在樱花隧道里看到了悠真——他长高了，表情变了，说"怎样都好吧"的时候眼神是空的。千纱的心在那一刻同时被两种情绪撕扯：重逢的喜悦，和"他已经不是从前的悠真了"的恐惧。她站在樱花纷飞中，叫出了那个七年没叫过的名字——"悠真くん"。然后立刻改口："……桐生くん，好久不见。"

**现在的距离**：千纱想靠近悠真，但不知道该以什么身份。青梅竹马？那已经是七年前的事了。同学？那和其他人有什么区别？她害怕太主动会吓到悠真，又害怕不主动就会像其他人一样变成"怎样都好吧"的对象。所以她选择了最笨的方式——一点一点地、小心翼翼地靠近，像小时候慢慢靠近一只受伤的小猫。

### 4.5 三层内心深度

| 层级 | 揭示内容 | 触发条件 |

| 第一层 | 千纱是悠真小时候的青梅竹马，七年后重逢，温柔得体但两人之间有微妙的距离 | 共通线自然揭示 |
| 第二层 | 千纱因为频繁搬家，每次刚交到朋友就要离开，已经不敢轻易投入感情。她的"得体"不是天性，而是反复练习出来的生存本能。和悠真独处时会变得粘人笨拙，因为她只有在悠真面前才敢卸下防备 | 共通线后期/个人线初期 |

### 4.6 核心矛盾

千纱的核心矛盾是**"想留下"与"习惯了离开"的拉扯**。

她比任何人都想留在悠真身边，但七年的搬家生活让她形成了一个根深蒂固的信念：**所有关系都是暂时的，所有人最终都会分开**。她不敢全力靠近，不是因为不在乎，而是因为太在乎——如果她像小时候那样毫无保留地依赖悠真，然后又不得不离开，那她承受不起第二次失去。

她把悠真当成"回忆中不可触碰的珍宝"，而不是"可以重新拥有的现实"。这就是她的青春之壁：**她把过去的悠真当作最珍贵的记忆保存着，却不敢相信现在的悠真也能成为她的未来**。

### 4.7 千纱的"拒绝"

在个人线中，当悠真终于主动靠近千纱、表达想和她在一起的意愿时，千纱会**拒绝**——不是因为她不喜欢悠真，恰恰相反，是因为她太喜欢了。

她的拒绝逻辑是："我可能又要搬走。如果我接受了你的心意然后又离开，你会比现在更受伤。我不想成为又一个让你觉得'怎样都好吧'的人。"

这个拒绝是千纱最痛苦的选择——她用伤害自己的方式来保护悠真。但千纱的"保护"是错误的——悠真需要的不是被保护，而是被选择。当悠真说出"就算你要走，我也想选择现在和你在一起"时，千纱的青春之壁才会真正突破。

### 5.1 外貌

暗红色的长发扎成低马尾——不是因为喜欢这个发型，而是低马尾不需要发饰，简单方便。校服洗得干干净净，袖口有反复缝补的痕迹，但缝得很整齐——她妈妈教她的，"东西旧不要紧，但要干净体面"。眼睛是深棕色的，看人的时候总是带着一点小心翼翼的观察，像是在确认"这个人会不会觉得我奇怪"。身材娇小，但搬货上架的时候力气意外地大——打工练出来的。笑起来的时候会微微低头，像是不太习惯被人看着笑。手上有细小的茧——打工搬货和长时间站立留下的痕迹，但她从不觉得这有什么不好。

### 5.2 性格

**表面：安静、礼貌、有点距离感的"普通女孩"**

茜不是那种让人一眼记住的人——她不张扬、不出众、不惹麻烦。对谁都客客气气，但不会主动凑上去聊天。她有一种恰到好处的距离感，不远不近，让人觉得"她人挺好的，但好像不太想被打扰"。实际上她只是不知道怎么自然地加入话题——别人聊新出的手机、周末去哪里玩、哪家店好吃，她插不上话，因为那些她都不太了解。

**内面：敏感、认真、有点倔强的少女**

茜的安静不是天性内向，而是怕说错话。她观察力很强，能从同学的便当推断家庭状况，但她从不说破——因为她知道被别人"看穿"是什么感觉。她很认真，打工从不迟到，作业从不拖延，因为"如果不用功，就真的什么都没有了"。她有点倔——别人帮她她会说"谢谢但不用了"，不是因为不需要，而是不想欠人情，因为欠了人情就得还，而她觉得自己没什么能还的。

**对悠真的特殊态度：被看见的安心**

悠真是第一个不是"同情"她而是"看见"她的人——他不是可怜她穷，而是注意到她搬货上架很利落、她打折布丁前会偷偷开心、她说"不用了"的时候其实很累。茜对悠真的方式是"慢慢靠近"——她不会主动闯入，但会在便利店给他多装一个饭团，会在图书室帮他留一本他想看的书。她害怕的不是被看见，而是被看见之后对方觉得"也就这样吧"。

**反差萌细节**：
- 在便利店打工时，常客进门她会小声说"欢迎回来"，但被发现后会立刻假装在整理货架
- 看到喜欢的书会无意识用手指在桌上描摹书名，像在抚摸，被发现会把手缩回去
- 便利店有打折的布丁时，眼睛会亮一下，然后假装很淡定地拿一个
- 打工结束后坐在便利店后门的台阶上，抬头看一会儿夜空才回家——那是她一天里唯一不赶时间的时刻
- 被人夸奖时会条件反射地说"没有啦"，耳朵尖会微微发红

### 5.3 日常行为

**便利店打工**：茜每天放学后在学校附近的便利店打工，从傍晚到晚上九点。她动作麻利，对常客的喜好记得很清楚——"桐生くん今天也买炒面面包吗？"。打工的时候她反而比在学校自在，因为便利店的规则很清楚——做什么、怎么做、做到什么程度，不需要猜别人的心思。

**学校便利店**：午休时间，茜在学校便利店帮忙。这是她在学校里最放松的时刻——穿着围裙站在货架后面，不需要和任何人社交，只需要把面包摆整齐、把牛奶放进冰箱、给来买东西的同学找零钱。悠真偶尔会趁午休来买饮料，茜会假装不经意地把他喜欢的口味放在最顺手的位置。

**图书室**：茜在图书室总是坐在角落，安静地看书或者整理图书。她喜欢图书室的原因很简单——安静、有空调、不用花钱。偶尔她会看着窗外出神，但有人经过就立刻低头假装在看书。

**午餐**：茜总是吃最便宜的便当，但她会把便当摆得整整齐齐——"吃得便宜不要紧，但要好好吃"。偶尔便利店的同事会给她快过期的饭团，她会开心地说"太好了，今天的晚饭有着落了"——说得很自然，没有自怜。

### 5.4 与悠真的关系

**相遇**：悠真经常在茜打工的便利店买东西，一来二去就熟悉了。茜记住了他的购买习惯，会在他常买的炒面面包上架后给他留一个。悠真一开始只是觉得"这个店员挺细心的"，后来偶尔在深夜路过，看到茜打工结束后坐在便利店后门台阶上抬头看夜空——那个安静侧脸和白天利落的店员判若两人。

**靠近**：茜对悠真的靠近是缓慢的——她会在他买东西时多聊两句，会在图书室帮他找书，会在他路过时说"今天有新到的炒面面包哦"。她不是在追求悠真，而是在他面前她可以稍微放松一点——因为他从不在意她穿什么、用什么，只是很自然地跟她说话。

**距离**：当悠真想帮她的时候，茜会本能地拒绝——"不用了，我自己可以的。"这不是逞强，而是她真的觉得自己可以。她从小就在学"什么都靠自己"，独立是她的生存方式，也是她的骄傲。但偶尔——真的只是偶尔——她也会想，有人帮忙的感觉，好像也不错。

### 5.6 核心矛盾

茜的核心矛盾是**"独立"与"依赖"的拉扯**。

她不是觉得自己不配，而是不想欠人情。从小看着妈妈一个人撑起整个家，她学会了一件事：自己的事自己解决，不要给别人添麻烦。这不是自卑，是她的生存哲学——也是她引以为豪的东西。

但问题在于，她把"独立"当成了唯一的生存方式。别人帮她，她第一反应是拒绝；别人关心她，她第一反应是"我没事"；别人想靠近她，她第一反应是"不用了，我自己可以"。她不是不需要帮助，而是不知道怎么接受帮助——因为接受帮助意味着承认"我一个人做不到"，而她一直告诉自己"我可以的"。

她的青春之壁是：**独立让她坚强，但也让她孤独——她把所有人都挡在了"我自己可以"的墙后面。**

### 5.7 茜的"拒绝"

在个人线中，当悠真想帮茜的时候，茜会反复拒绝——不是因为她不领情，而是因为她不知道怎么接受。

她的拒绝逻辑是："我自己可以做到的，不用麻烦你。——不是逞强，是真的可以。只是……你为什么要帮我？我们只是……便利店的店员和客人而已吧？"

这个拒绝不是在推开悠真，而是在确认——你帮我是因为你愿意，还是因为你觉得我可怜？如果是后者，她不需要；如果是前者，她不知道该怎么回应。

小春在母亲去世后跟随父亲。父亲再婚后，她有了新的母亲和——一个哥哥。从第一天起，悠真就对她很好：帮她搬行李、带她认路、在她害怕打雷的夜晚安慰她。小春很快发现，自己对悠真的感情不只是"哥哥"。但"兄妹"这个身份像一道无形的墙——她不敢越界，也害怕越界后会失去现在拥有的一切。

### 6.3 小春的日常

小春是桐生家真正的"管家"。每天比悠真早起半小时做早餐，校服永远熨得平整，便当营养均衡且会根据悠真前一天的状态调整菜单。她不是被迫做这些——"照顾哥哥"是她最幸福的时刻。悠真的每一个习惯她都记在心里：思考时无意识转笔、紧张时摸左手腕、说"怎样都好吧"的时候其实最在意。她比悠真自己更了解悠真。

放学后和周末，小春在站前商店街的咖啡店"star"打工。她穿着围裙在吧台后冲咖啡的样子，和在家穿围裙做饭的样子，是同一个人，却是两种完全不同的氛围——在咖啡店的小春更放松，偶尔会和熟客聊几句，甚至会露出在家很少见的笑容。悠真有时会去咖啡店自习，名义上是"找个安静的地方看书"，实际上小春知道他是来看自己的——但她从不点破，只是默默把他的咖啡换成他最喜欢的口味。

### 6.4 小春的双面性

在家人面前，小春是安静乖巧的妹妹，话不多，总是微笑着做自己的事。但和悠真独处时，她会变得坦率——悠真说冷笑话时她会面无表情地吐槽，偶尔冒出让人哭笑不得的天然发言。这种反差是只有悠真才能看到的。在其他角色的路线中，小春是那个能冷静指出悠真问题的人——"哥哥，你在逃避吧""你其实很想靠近她，对吗？"。她用自己的理性帮助哥哥，即使这意味着把他推向别人。

### 6.5 核心矛盾

小春的理性是她最大的牢笼。她比任何人都清楚"不应该喜欢哥哥"——重组家庭的微妙、社会的眼光、可能失去一切的后果。她不是不敢告白，而是理性地选择了不告白。但理性无法消灭感情，只会让感情在沉默中发酵成痛苦。她最害怕的不是被拒绝，而是告白之后连"妹妹"的位置都保不住。

### 7.1 性格

典型的 galgame 好兄弟——永远精力充沛、永远自来熟、永远在主角身边转悠。纯粹的气氛调节器：有他在永远不会冷场，有他在的午休永远不会无聊。

核心功能：**让悠真不至于彻底孤立**。悠真习惯旁观、习惯说"怎样都好吧"，但信不吃这套——直接一巴掌拍在悠真背上说"走啦，吃饭"，不等回答就把人拽走。他不是在关心悠真的心理状态，他只是觉得"一个人吃饭多没意思"，而这种不刻意的陪伴恰恰是悠真最需要的。

信的情报网覆盖整个年级——谁在追谁、哪个老师最严、哪家店打折最快，他全知道。情报来源永远是"我听说的"，但准确率高得离谱。

### 7.2 日常行为

- 午休端着便当理所当然地坐到悠真旁边，边吃边聊八卦，悠真基本不搭话但他一个人也能说完整顿饭
- 放学后经常拽悠真去商店街逛，嘴上说"陪我去买东西"，实际上是想让悠真多出门
- 对女生话题异常热衷，撺掇悠真"你倒是主动点啊"，但自己完全没有恋爱经验
- 口头禅："我有个绝密情报"和"信我准没错"
- 偶尔说出意外有道理的话，说完自己先愣住："我刚才是不是说了什么很帅的话？"

### 7.3 与悠真的关系

高一第一天就单方面认定悠真是死党。从不过问悠真的家庭状况——不是体贴，而是他根本没往那方面想。对他来说，悠真就是那个坐在窗边发呆、偶尔接一句吐槽能让他笑半天的家伙。他不知道悠真在家里的疏离感，只知道"这家伙一个人待着的时候表情有点寂寞"，所以他会凑过去。

在个人线中，信偶尔充当"推手"——用他特有的方式给悠真制造机会，比如故意把悠真推到女主角面前说"他有话跟你说"。
