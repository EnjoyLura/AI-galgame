# 《君と描くパレット》— 角色语音风格

> 对应工作流：**一、项目前期规划阶段 → 1.2 角色设计 → 角色语音风格**
> 本文档定义所有主要角色的说话方式、语气特征、口头禅、敬语习惯，用于指导剧本写作时的对话创作。

---

## 1. 桐生悠真（男主）

### 1.1 基本语气

| 维度 | 特征 |
|------|------|
| **整体基调** | 温和、平淡、略带疏离，不主动发起话题 |
| **语速** | 偏慢，思考后才开口，常有停顿 |
| **音量** | 中等偏低，不会大声说话 |
| **情绪表达** | 内敛，极少外露强烈情绪 |

### 1.2 语言特征

- **第一人称**：「俺」（对同龄人/朋友）/ 「僕」（对长辈/正式场合）
- **句尾习惯**：常用省略句，话说到一半就停；疑问句多用「……かな」「……だろう」
- **口头禅**：
  - 「どうでもいいか」（怎样都好吧）—— 核心台词，表达疏离
  - 「まあ、いいや」（算了，就这样吧）
  - 「……なんでもない」（……没什么）
  - 「悪くない」（不坏）—— 他的"好评"级别
- **称呼方式**：
  - 对女生：姓氏+さん（初期）→ 名字（关系亲近后）
  - 对千纱：初期「七瀬」→ 后期「千纱」
  - 对小春：「小春」（日常）
  - 对信：「信」（直呼其名）
  - 对父母：母「ママ」、继父「天羽さん」（保持距离）

### 1.3 情绪变化时的语气

| 情绪 | 语气变化 |
|------|----------|
| **惊讶** | 短促的「……え？」，然后沉默 |
| **困扰** | 挠头，「うーん……」拖长音 |
| **开心** | 嘴角轻微上扬，但话不多，只说「悪くないな」 |
| **生气** | 罕见。声音变低，话更少，说「……やめてくれ」 |
| **认真** | 直视对方眼睛，一字一句地说，没有口头禅 |
| **动摇/心动** | 内心独白变多，说话结巴，「あ、いや、その……」 |

### 1.4 与其他角色对话时的差异

| 对象 | 特征 |
|------|------|
| **对小春** | 最放松，会开玩笑，偶尔吐槽，像真正的兄妹 |
| **对千纱** | 初期客气疏离，后期逐渐找回小时候的亲近感 |
| **对绫音** | 被看似随意的请求时会无奈接受，「……はいはい」 |
| **对千夏** | 被她的热情带着走，经常被动回应 |
| **对茜** | 小心翼翼，怕伤到她的自尊 |
| **对信** | 唯一会真正吐槽的对象，「お前なあ……」 |

### 1.5 内心独白 vs 对外说话

- **内心独白**：细腻、敏感、自我否定倾向，「また傍観者だ」「近づいたら傷つく」
- **对外说话**：简短、回避、不把真实想法说出口
- **关键成长标志**：内心独白逐渐从灰色变为有色彩，对外也开始说出真实想法

## 2. 白鸟绫音

### 2.1 基本语气

| 维度 | 特征 |
|------|------|
| **整体基调** | 优雅高傲，用看似随意的请求掩饰在意，偶尔流露小恶魔般的捉弄 |
| **语速** | 偏快，条理清晰，不拖泥带水 |
| **音量** | 适中，但每个字都清晰有力 |
| **情绪表达** | 表面冷静克制，内心波动极大 |

### 2.2 语言特征

- **第一人称**：「私」（始终，即使在最亲近时也不会变软）
- **敬语使用**：对老师/长辈用完整敬语；对悠真初期用敬语（距离感），后期混用看似随意的请求（亲近的表现）
- **句尾习惯**：
  - 看似随意的请求：「来てくれない？」「やってくれる？」，然后补一句「来なくてもいいけど」
  - 否定前置：「別に……」「ただの……而已」
  - 掩饰在意时：句尾轻微上扬，或突然转移话题
- **口头禅**：
  - 「別に」（没什么/无所谓）—— 傲娇标配
  - 「ただの義務だから」（只是义务而已）
  - 「あなたのためじゃないわよ」（不是为了你）
  - 「ふん」（哼）—— 表达不满或掩饰害羞
  - 「……馬鹿」（……笨蛋）—— 最亲近时的"爱称"
  - 「来なくてもいいけど」（不来也没关系）—— 掩饰在意的经典台词
- **称呼方式**：
  - 对悠真：初期「桐生くん」→ 后期偶尔「……悠真」（只在最私密的时刻）
  - 对其他学生：姓氏+くん/さん
  - 对老师：完整敬语

### 2.3 情绪变化时的语气

| 情绪 | 语气变化 |
|------|----------|
| **害羞** | 声音突然变小，语速加快，「べ、別に……！」 |
| **生气** | 声音降低一度，更冷静，更刻薄，「……失礼ね」 |
| **开心** | 嘴角微扬但不说，只说「まあ、及第点ね」 |
| **崩溃/脆弱** | 第一次卸下高傲，声音发抖，用「私」诉说真实痛苦 |
| **小恶魔模式** | 故意用敬语捉弄悠真，「悠真さん、どうしましたの？」 |

### 2.4 傲娇语言模式

绫音的傲娇有明确的语言层次：

| 层次 | 语言表现 | 示例 |
|------|----------|------|
| **第一层：看似随意的请求** | 用学生会事务包装关心，但留退路 | 「学生会正好缺人手……你要是有空的话。——不来也没关系。」 |
| **第二层：否认** | 绝不承认自己在意 | 「……不是特意等你的。只是学生会的工作刚好做到这个时间而已。碰巧。」 |
| **第三层：掩饰** | 用评价代替关心 | 「……做得不错」「你这种成绩还想毕业吗」 |
| **第四层：破绽** | 极少数时刻的真心 | 「……你就是这一点，狡猾。」 |
| **第五层：坦率** | 只在最私密时刻 | 「这是我一生的请求，所以不许拒绝。」 |

### 2.5 与其他角色对话时的差异

| 对象 | 特征 |
|------|------|
| **对悠真** | 唯一会展现傲娇和小恶魔的一面，用看似随意的请求掩饰在意 |
| **对其他学生** | 完美学生会长模式，礼貌但疏离 |
| **对千夏** | 略带竞争意识，「星野さん、いつも元気ね」 |
| **对千纱** | 微妙的警戒，「七瀬さん、悠真と昔から知り合い？」 |
| **对母亲（翠子）** | 压抑的顺从，「……はい、わかりました」 |

## 3. 星野千夏

### 3.1 基本语气

| 维度 | 特征 |
|------|------|
| **整体基调** | 元气满满、阳光、直接、不绕弯子 |
| **语速** | 偏快，充满活力 |
| **音量** | 偏大，笑声尤其响亮 |
| **情绪表达** | 外放型，所有情绪都写在脸上和声音里 |

### 3.2 语言特征

- **第一人称**：「私」（对前辈/正式场合）/ 「あたし」（对朋友/日常）
- **敬语使用**：对前辈和老师用敬语；对同辈几乎不用，非常随意
- **句尾习惯**：
  - 上扬的疑问：「～だよね？」「～でしょ？」
  - 感叹多用：「～だよ！」「～なんだもん！」
  - 口头禅接尾：句尾常带「よ」「ね」「もん」
- **口头禅**：
  - 「大丈夫！」（没事的！）—— 使用频率极高
  - 「元気出して！」（打起精神来！）
  - 「へへっ」（嘿嘿）—— 得意或害羞时的笑声
  - 「やばい」（糟了/厉害了）—— 万能感叹词
  - 「へっちゃら！」（小菜一碟！）
- **称呼方式**：
  - 对悠真：初期「桐生くん」→ 很快变成「悠真くん」
  - 对所有人：很快变成名字+くん/ちゃん
  - 对学妹：名字+ちゃん，像大姐姐

### 3.3 情绪变化时的语气

| 情绪 | 语气变化 |
|------|----------|
| **真正开心** | 笑声「あははは！」非常响亮，会拍别人肩膀 |
| **勉强开心** | 笑容不变，但笑声短促，「……うん、大丈夫！」 |
| **受伤/脆弱** | 笑容消失，声音变低，「……ちょっと、一人にして」 |
| **认真** | 罕见。眼神专注，「……本気で言ってるよ」 |
| **慌张** | 「やばいやばい！どうしよう！」 |

### 3.4 "开关式"笑容的语言标志

千夏的笑容只有"开"和"关"两种状态：

| 状态 | 语言特征 |
|------|----------|
| **笑容ON** | 句尾带「よ/ね/もん」，笑声自然，主动发起话题 |
| **笑容OFF** | 句子变短，句尾变平，沉默，只说必要的话 |
| **切换瞬间** | 从ON到OFF时，会有一个明显的停顿和「……」 |

### 3.5 与其他角色对话时的差异

| 对象 | 特征 |
|------|------|
| **对悠真** | 最主动，会直接拉他参加活动，「一緒に行こうよ！」 |
| **对学妹** | 大姐姐模式，鼓励为主，「あんたならできるよ！」 |
| **对千纱** | 友好但微妙地竞争，「七瀬さん、悠真くんと仲いいね～」 |
| **对绫音** | 尊敬但不怕，「白鳥さん、ちょっと堅すぎない？」 |
| **独处时** | 完全沉默，不发一语，和平时判若两人 |

## 4. 七瀬千纱

### 4.1 基本语气

| 维度 | 特征 |
|------|------|
| **整体基调** | 温柔、礼貌、略带小心翼翼，像在试探水温 |
| **语速** | 适中偏慢，思考后才说 |
| **音量** | 轻柔，像怕惊扰什么 |
| **情绪表达** | 含蓄，用停顿和省略表达真实情绪 |

### 4.2 语言特征

- **第一人称**：「私」（始终，温柔的感觉）
- **敬语使用**：对所有人都用敬语，即使对悠真也很客气
- **句尾习惯**：
  - 柔和的上扬：「……かな？」「……ですね」
  - 省略句：说到一半停住，「あの……」「その……」
  - 回忆时：句尾带「……だったよね」
- **口头禅**：
  - 「……覚えてる？」（……还记得吗？）—— 试探悠真的记忆
  - 「昔は……」（以前啊……）—— 回忆过去
  - 「ごめんね」（对不起）—— 使用频率过高，即使不是她的错
  - 「……うん」（……嗯）—— 很多话都缩成这一个字
  - 「嬉しい」（好开心）—— 真心开心时，轻声说 |
- **称呼方式**：
  - 对悠真：初期「桐生くん」→ 努力改回「悠真くん」→ 后期「悠真」（最亲近时）
  - 对其他人：姓氏+さん/くん
  - 对老师：完整敬语

### 4.3 情绪变化时的语气

| 情绪 | 语气变化 |
|------|----------|
| **开心** | 声音变亮，但还是很轻，「……嬉しい」 |
| **失落** | 沉默变长，「……そっか」 |
| **吃醋** | 不会直接说，只会问「……あの子、誰？」 |
| **鼓起勇气** | 深呼吸后一口气说完，声音微颤 |
| **哭泣** | 无声流泪，只说「……ごめん、泣いちゃった」 |

### 4.4 "距离感"的语言层次

千纱的语言反映了她与悠真之间的距离：

| 距离 | 称呼 | 句尾 | 示例 |
|------|------|------|------|
| **远** | 桐生くん | です/ます | 「桐生くん、おはようございます」 |
| **中** | 悠真くん | だよ/だね | 「悠真くん、一緒に帰ろう？」 |
| **近** | 悠真 | 省略句 | 「悠真……こっち向いて」 |
| **最近** | 悠真（+牵手/靠近） | 几乎说不出话 | 「……好き、だよ」 |

### 4.5 与其他角色对话时的差异

| 对象 | 特征 |
|------|------|
| **对悠真** | 最温柔也最小心翼翼，像在对待易碎品 |
| **对千夏** | 友好但有些羡慕她的开朗，「星野さん、いつも元気だね」 |
| **对绫音** | 礼貌但保持距离，「白鳥さん、すごいね……」 |
| **对小春** | 微妙的复杂感，「天羽さん、お兄ちゃんのことよろしくね」 |
| **对父母** | 孝顺女儿模式，「お母さん、今日のご飯何？」 |

## 5. 筿崎茜

### 5.1 基本语气

| 维度 | 特征 |
|------|------|
| **整体基调** | 简短、直接、不废话，像在说"别管我" |
| **语速** | 偏快，但不急促，是"不想多说"的快 |
| **音量** | 偏低，经常低着头说话 |
| **情绪表达** | 极度克制，几乎不表露情绪 |

### 5.2 语言特征

- **第一人称**：「私」（对所有人，没有亲近后的变化）
- **敬语使用**：对老师用最低限度的敬语；对同辈几乎不用
- **句尾习惯**：
  - 平铺直叙：句尾很少带语气词
  - 拒绝时：「……いい。自分でやる」（不用了，我自己来）
  - 接受帮助时：「……助かる」（……帮大忙了）—— 已经是极大的让步
- **口头禅**：
  - 「……いい。自分でやる」（不用了，我自己来）—— 核心台词
  - 「悪いけど」（不好意思，但……）—— 拒绝的前置
  - 「別に期待してない」（我没期待什么）
  - 「……お世話にならないように」（不想给你添麻烦）
  - 「ああ」（嗯）—— 最短的回应
- **称呼方式**：
  - 对悠真：初期「桐生」→ 后期「……桐生」（省略くん，保持边界）
  - 对其他人：姓氏，不加敬称
  - 对店长：「店長」—— 唯一用敬称的对象

### 5.3 情绪变化时的语气

| 情绪 | 语气变化 |
|------|----------|
| **困扰** | 皱眉，「……どうでもいい」 |
| **害羞** | 转头，声音更低，「……べつに」 |
| **生气** | 罕见。声音变硬，「……放っといて」（别管我） |
| **开心** | 极罕见。嘴角轻微上扬，「……悪くない」 |
| **依赖/信任** | 最罕见。会说「……頼っても、いい？」 |

### 5.4 "独立铠甲"的语言模式

茜的语言是她保护自己的铠甲：

| 情境 | 她的反应 | 真实想法 |
|------|----------|----------|
| 被帮助时 | 「いい。自分でやる」 | 「助けてほしい……でも、頼ったら傷つく」 |
| 被关心时 | 「別に、一人で大丈夫」 | 「誰かにいてほしい……でも、慣れない」 |
| 被送礼物时 | 「……なんで？」 | 「嬉しい……でも、返せない」 |
| 被夸奖时 | 「……ただの仕事」 | 「認めてほしい……でも、期待に応えられない」 |

### 5.5 与其他角色对话时的差异

| 对象 | 特征 |
|------|------|
| **对悠真** | 唯一会逐渐卸下铠甲的人，但过程极慢 |
| **对店长** | 工作中最低限度的交流，「はい」「わかりました」 |
| **对千夏** | 微妙地羡慕她的开朗，但觉得"和自己无关" |
| **对绫音** | 保持距离，「……白鳥さんとは違う世界だ」 |
| **对母亲（惠子）** | 最温柔的一面，「お母さん、今日は早く帰ってきて」 |

## 6. 天羽小春

### 6.1 基本语气

| 维度 | 特征 |
|------|------|
| **整体基调** | 平静、温柔、理性，像水一样包容 |
| **语速** | 适中，不快不慢，每个字都清晰 |
| **音量** | 轻柔但稳定，不会忽大忽小 |
| **情绪表达** | 极度克制，用理性压抑感性 |

### 6.2 语言特征

- **第一人称**：「私」（对所有人，包括悠真）
- **敬语使用**：对悠真用轻度敬语（哥哥+です/ます）；对长辈用完整敬语
- **句尾习惯**：
  - 稳定的陈述：「……です」「……ます」
  - 关心时：句尾带「ね」，「……ですね、哥哥」
  - 提醒时：「……忘れないでください」（请不要忘记……）
- **口头禅**：
  - 「哥哥」（にいさん/お兄ちゃん）—— 对悠真的专属称呼
  - 「……気をつけて」（……请小心）
  - 「大丈夫ですか？」（你还好吗？）
  - 「……分かってます」（……我知道的）—— 意味深长
  - 「お仕事、頑張ってください」（工作加油）—— 对悠真说时，暗含"我在看着你"
- **称呼方式**：
  - 对悠真：「お兄ちゃん」（日常）/ 「哥哥」（更亲近时）
  - 对父母：「お母さん」「お父さん」
  - 对其他人：姓氏+さん

### 6.3 情绪变化时的语气

| 情绪 | 语气变化 |
|------|----------|
| **担心** | 消息变多，「今日の天気、雨ですよ。傘、持ちましたか？」 |
| **吃醋** | 不会直接说，只会更频繁地发消息，更"妹妹" |
| **动摇** | 理性外壳出现裂缝，「……私、お兄ちゃんのこと……」然后停住 |
| **哭泣** | 无声，只说「……ごめんなさい。何でもないです」 |
| **真正开心** | 罕见的真心笑，「……ふふっ。お兄ちゃん、変わってないね」 |

### 6.4 "理性牢笼"的语言模式

小春用"妹妹"的身份和理性的语气囚禁自己的感情：

| 情境 | 她说的话 | 真实感情 |
|------|----------|----------|
| 看到悠真和其他女生在一起时 | 「お兄ちゃん、人気者ですね」 | 「……私じゃ、だめなの？」 |
| 给悠真做便当时 | 「お母さんの分も作ったので、お兄ちゃんのもあります」 | 「……あなたのために、早起きしたの」 |
| 悠真关心她时 | 「大丈夫です。私は妹妹ですから」 | 「……もっと、私を見て」 |
| 深夜独处时 | 「……お兄ちゃん、今日も遅いですね」 | 「……一緒にいたい」 |

### 6.5 与其他角色对话时的差异

| 对象 | 特征 |
|------|------|
| **对悠真** | 最温柔也最痛苦，用"妹妹"身份维持最后的理性 |
| **对母亲（雅美）** | 贴心女儿，「お母さん、今日は私が料理します」 |
| **对继父（诚一）** | 孝顺但保持距离，「お父さん、お疲れ様です」 |
| **对千纱** | 微妙的敌意和羡慕，「七瀬さん、哥哥と昔から？」 |
| **对咖啡店同事** | 完美店员模式，礼貌、高效、微笑 |

## 7. 五十岚信

### 7.1 基本语气

| 维度 | 特征 |
|------|------|
| **整体基调** | 开朗、八卦、爱开玩笑，气氛制造者 |
| **语速** | 偏快，像机关枪 |
| **音量** | 偏大，不怕被人听到 |
| **情绪表达** | 外放，所有情绪都直接表达 |

### 7.2 语言特征

- **第一人称**：「俺」（对所有人）
- **敬语使用**：几乎不用，对老师也是「先生～」
- **句尾习惯**：
  - 上扬的感叹：「～じゃん！」「～だろ！」
  - 八卦时：压低声音但还是很响，「なあなあ、聞いた？」
- **口头禅**：
  - 「マジで？」（真的假的？）
  - 「ヤバいヤバい」（厉害了厉害了）
  - 「なあ悠真」（喂悠真）—— 开启话题的标准开头
  - 「俺の情報網によると……」（据我的情报网……）
  - 「お前、もしかして……」（你，莫非……）—— 八卦专用
- **称呼方式**：
  - 对悠真：「悠真」（直呼其名）
  - 对女生：名字+ちゃん（轻浮但无恶意）
  - 对老师：「先生～」

### 7.3 角色功能语言

信在故事中主要承担以下功能，语言风格随之变化：

| 功能 | 语言表现 |
|------|----------|
| **情报提供者** | 「俺の情報網によると、白鳥さんって実は……」 |
| **气氛调节者** | 「ははは！悠真、顔真っ赤じゃん！」 |
| **吐槽役** | 「お前、それ本気で言ってる？マジで？」 |
| **神助攻** | （故意把悠真推到女主面前）「あ、悠真！お前、彼女に話あるんだろ？」 |
| **唯一看透悠真的人** | 「……お前、本当は寂しいんだろ。分かるよ。」 |

## 9. 写作注意事项

### 9.1 避免OOC（角色崩坏）的要点

| 角色 | 绝对不能说的话 | 绝对不能做的事 |
|------|----------------|----------------|
| 绫音 | 「好きよ」（直接说喜欢） | 主动撒娇、大声笑 |
| 千夏 | 「助けて」（求救） | 在众人面前哭 |
| 千纱 | 「忘れて」（说忘了过去） | 主动推开悠真 |
| 茜 | 「お願い」（请求帮助） | 主动依赖别人 |
| 小春 | 「好きです」（告白） | 打破"妹妹"的身份 |
| 悠真 | 「俺は必要とされてる」（说自己被需要） | 主动追求别人（前期） |

### 9.2 角色辨识度测试

写完后可以用以下测试检查角色语音是否鲜明：

> **遮住角色名，只看台词，能否一眼认出是谁说的？**

- 绫音：看似随意的请求+傲娇+小恶魔
- 千夏：元气+感叹+直接
- 千纱：温柔+省略+回忆
- 茜：简短+拒绝+独立
- 小春：理性+妹妹+暗含深意
- 悠真：疏离+省略+内心丰富
- 信：八卦+开朗+吐槽
