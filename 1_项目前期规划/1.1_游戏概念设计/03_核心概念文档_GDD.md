# 《君と描くパレット》— 核心概念文档 (GDD)

## 1. 游戏核心机制

### 1.1 路线分支系统

- 不设好感度数值，通过**固定选项**直接决定剧情走向
- 共通线中每个关键选择点明确指向不同路线，玩家选择"和谁在一起"即走向对应角色
- **烟花大会**是共通线最终分歧点，此选择锁定进入哪条个人线
- 若共通线中未累计触发任何角色足够数量的关键flag，烟花大会自动进入 **Bad End（单身线）**
- 个人线中每个选择直接决定剧情分支，不累积数值，选项本身即结果
- 路线分支以**flowchart**形式可视化展示，玩家可在流程图中回顾和跳转已解锁/未解锁的剧情节点

> 路线分支的详细flowchart和选择点设计见 → [选择点设计](../../2_叙事架构设计阶段/03_选择点设计.md)

### 1.2 日常选择系统

- 每天的自由行动时间，选择去哪里/和谁互动
- 不同选择触发不同日常事件，同时决定后续可用的剧情分支
- 选择不是"累积好感"，而是"打开/关闭"对应的剧情路径
- 关键flag节点包括：学园祭选择、田径联赛观赛、自由活动时间选择、学习会场所选择、周末活动选择等节点
- 若所有角色关键flag节点数量均不足，烟花大会结束后将进入Bad End

### 1.3 关键事件系统

- 故事中有固定的关键事件，按时间顺序排列：
  - **入学式**（4月初）：角色初遇/重逢
  - **社团招新**（4月中）：展现角色社团身份
  - **学园祭**（6月初）：全角色大型事件，各角色有专属选择时刻
  - **田径联赛**（6月上旬）：千夏线转折前置事件
  - **期中考试+学习会**（6月中）：全员强制事件
  - **烟花大会**（7月初）：共通线最终分歧点，Bad End触发点
- 关键事件中与当前路线角色触发专属剧情
- 关键事件是不可跳过的关键节点
- 烟花大会是共通线最终分歧点，此选择锁定进入哪条个人线

### 1.4 "青春之壁"系统

- 每个角色都有一个核心心理困境，称为"青春之壁"
- 青春之壁分为三层，随剧情推进逐层揭示
- 突破第三层需要在关键选择点做出正确选择
- 突破青春之壁是进入Good End的必要条件

**各角色青春之壁三层结构**：

| 角色 | 第一层（表面） | 第二层（真相） | 第三层（核心） |
|------|-------------|-------------|-------------|
| 绫音 | "我必须完美"——在所有人面前维持优等生形象 | "母亲的爱让我窒息"——被母亲的控制欲束缚，无法追求真正的热爱 | "如果我不完美，我还值得被爱吗？"——对无条件被爱的渴望与恐惧 |
| 千夏 | "我没事！"——永远用笑容面对所有人 | "如果我不跑了，还有人记得我吗？"——用运动成绩定义自我价值 | "我不允许自己脆弱"——害怕一旦示弱就会被抛弃 |
| 千纱 | "好久不见"——用礼貌的距离掩饰七年的空白 | "你还记得小时候的事吗？"——害怕只有自己还在意过去 | "如果再次靠近，会不会再次失去？"——对离别的创伤 |
| 茜 | "不用了，我自己可以"——拒绝一切帮助 | "我不配拥有普通学生的生活"——贫困带来的劣等感 | "依赖别人就会被抛弃"——父亲离开的创伤 |
| 小春 | "我是妹妹"——用身份框定关系 | "如果我不是妹妹，你会看我吗？"——对身份的憎恨与依赖 | "打破这层关系，我们会失去一切吗？"——对家庭崩坏的恐惧 |

**各路线青春之壁突破事件**：

| 角色 | 突破事件 | 悠真的关键行动 |
|------|----------|---------------|
| 绫音 | 画展投稿获奖，用结果说服母亲 | 帮绫音找到投稿机会，但让她自己赢 |
| 千夏 | 受伤后悠真探病，看到真实的房间 | 在她最脆弱时没有离开，也没有同情 |
| 千纱 | 旧地重游告白——"七年前没说完的话" | 主动握住她的手，回应七年的等待 |
| 茜 | 茜主动邀请看电影——第一次为别人花钱 | 接受她的邀请，肯定她的主动 |
| 小春 | 父母加班夜晚——"就一会儿" | 打破沉默，不再用"兄妹"做借口 |

### 1.5 "悠真成长"系统

悠真作为男主，同样有自己的"青春之壁"——重组家庭中的微妙距离感。母亲雅美开朗活泼，继父诚一稳重可靠、时常鼓励他，这个家不缺温暖。但雅美和诚一越幸福，悠真越觉得自己像个"前一段婚姻的遗留物"——不是被冷落，而是找不到属于自己的位置。他有渴望被人理解、被人关心、被人需要、被人认可的烦恼，但是觉得自己很普通，大多时候扮演者旁观者。他的成长与每条路线的女主紧密绑定，体现"互相救赎"的核心。

| 成长阶段 | 悠真变化 | 对应台词变化 |
|----------|----------|-------------|
| 局外人 | 旁观者心态 | "怎样都好吧。" |
| 微光 | 开始在意某个人的事 | "她……好像在勉强自己。" |
| 着色 | 主动想要帮助对方 | "我想为她做点什么。" |
| 归属 | 感受到被需要的重量 | "原来被需要是这种感觉。" |
| 觉醒 | 做出勇敢的选择，不再逃避 | "这次，我不会再退缩了。" |
| 有色 | 找到自己的归属和意义 | "我不再是局外人了。" |

> 成长阶段的具体触发条件见 → [选择点设计](../../2_叙事架构设计阶段/03_选择点设计.md)

**各路线中悠真的成长主题**：

| 路线 | 悠真学到的 |
|------|-----------|
| 绫音线 | 追求自我的勇气 |
| 千夏线 | 允许自己被照亮 |
| 千纱线 | 珍惜不放手 |
| 茜线 | 被信任的重量 |
| 小春线 | 打破理性与身份的枷锁追求真爱 |

### 1.6 Bad End 设计

**触发条件**：烟花大会是共通线与个人线的最终分界点。若玩家在共通线中未与任何角色建立足够深入的关系（即未触发任何角色足够数量的关键flag），烟花大会结束将自动进入Bad End。

**Bad End 内容**：
- 烟花大会当晚，悠真独自一人来到会场
- 他在人群中看到各女主角——绫音和学生会成员在一起、千夏和田径部后辈们在一起、千纱和家人在一起、茜在便利店打工、小春和父母在一起
- 没有人注意到他，他也没有走向任何人
- 烟花升空，悠真独自站在远离人群的角落看完
- 结尾独白：「第一学期结束了。什么也没有改变。我还是那个旁观者。」
- 画面渐暗，回到标题画面

**设计意图**：
- Bad End不是惩罚，而是对"不选择"的叙事回应——悠真的"局外人"困境在无人选择他、他也不选择任何人的情况下被固化
- 与五条路线的Good End形成对比：每条Good End中悠真都找到了"色彩"，而Bad End中他依然是"无色"
- 激励玩家重新开始、做出不同的选择

## 2. 美术风格定义

| 维度 | 方向 |
|------|------|
| 整体风格 | 日系二次元，明亮清新的校园画风，参考《突然*恋人》/《苍之彼方的四重奏》 |
| 色调倾向 | 整体明亮温暖，以天蓝、白色、淡粉为主调 |
| 立绘风格 | 高精度立绘，注重表情细腻度和服装质感，每角色12种以上表情差分 |
| 背景风格 | 日系校园写实背景，注重光影和氛围感，天空表现力强 |
| CG风格 | 恋爱氛围感构图，注重角色表情和肢体语言，关键告白场景全屏CG |
| UI风格 | 清新简约风，主色调为天蓝+白色+淡粉 |
