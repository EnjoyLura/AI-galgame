## 1. 男主：桐生悠真

| 项目 | 内容 |
|------|------|
| 身份 | 星见丘学园二年级，重组家庭的长子 |
| 家庭 | 母亲离婚后带他改嫁，继父带了一个女儿（天羽小春） |
| 性格 | 温和但疏离，习惯旁观，不主动靠近任何人 |
| 困境 | 在"家"中找不到归属——不是不被爱，而是不知道自己该站在哪个位置 |
| 核心台词 | "怎样都好吧。" |
| 社团 | 未加入（故事开始时） |

### 1.1 悠真的内心困境

悠真的问题来自重组家庭中的"局外人感"——不是被虐待或被忽视，而是那种微妙的、说不出口的疏离。

| 困境层次 | 表现 | 内心独白 |
|----------|------|----------|
| 局外感 | 在家里总觉得自己是"桐生家的"，而继父和妹妹是"天羽家的" | "我好像住在一个很温暖的旅馆里。" |
| 无归属 | 不知道自己该站在哪个位置——不是亲生儿子，也不是完全的外人 | "妈有了新的家庭，我算什么呢？" |
| 逃避亲近 | 对继父客气疏离，对妹妹保持"哥哥"的距离，不敢真正靠近任何人 | "太近了会受伤吧——反正怎样都好。" |
| 无方向 | 没有梦想、没有目标，觉得未来怎样都无所谓 | "连自己属于哪里都不知道，谈什么将来。" |
| 自我否定 | 觉得自己的存在是母亲"再婚的包袱" | "如果妈没有带着我，她大概会更幸福吧。" |

**核心矛盾**：悠真渴望归属、渴望被真正需要，但重组家庭的微妙距离让他不敢靠近任何人。他不是不想要，而是害怕靠近之后发现自己依然是局外人。这种恐惧让他选择了旁观——只要不靠近，就不会受伤。

**救赎契机**：当悠真开始帮助五位少女面对她们的问题时，他第一次感受到了"被需要"的重量。而当他发现自己真的可以成为某个人"不可替代的存在"时，他才明白——归属不是被给予的，而是自己选择的。

### 1.2 各路线中悠真的成长主题

| 路线 | 悠真学到的 | 对应悠真的困境突破 |
|------|-----------|-------------------|
| 綾音线 | 追求自我的勇气 | 从"在别人家里不敢做自己"→"有勇气选择自己想要的" |
| 千夏线 | 允许自己被照亮 | 从"害怕靠近会受伤"→"不靠近才是最大的伤害" |
| 千紗线 | 珍惜不放手 | 从"怎样都好吧"→"不想失去就要说出来" |
| 茜线 | 看见别人看不见的价值 | 从"觉得自己是包袱"→"每个人都有不可替代的价值" |
| 小春线 | 打破理性与身份的枷锁 | 从"被身份定义"→"选择自己想成为的人" |

| 代表色 | 藤紫色 |

### 2.1 三层内心深度

| 层级 | 揭示内容 | 触发条件 |
|------|----------|----------|
| 第一层 | 綾音是完美的学生会长，优雅从容，但偶尔流露出疲惫 | 共通线自然揭示 |
| 第二层 | 綾音秘密地画漫画，梦想是成为漫画家而非母亲期望的"名门淑女" | 共通线后期/个人线初期 |

| 色彩意象 | 优雅、高贵、被束缚 |
| 服装倾向 | 制服穿得一丝不苟，私服偏素雅 |

## 3. 星野千夏

| 项目 | 内容 |
|------|------|
| 萌属性 | 阳光向日葵/照亮所有人的少女 |
| 身份 | 星见丘学园二年级，独居 |
| 家庭 | 父母常年海外工作，独居 |
| 代表色 | 天蓝色 |
| 社团 | 田径部 |
| 青春之壁 | 她照亮所有人，却从不允许自己脆弱——"总是坚强"本身就是她的牢笼 |

### 3.1 三层内心深度

| 层级 | 揭示内容 | 触发条件 |
|------|----------|----------|
| 第一层 | 千夏是班上最开朗的人，总是第一个打招呼、第一个帮忙、第一个笑。她看起来自由自在，独居也过得很好 | 共通线自然揭示 |
| 第二层 | 千夏的阳光不是面具，但有一个盲区——她照顾所有人的情绪，唯独不允许自己示弱。她从不诉苦、从不求助、从不让人看到她疲惫的样子 | 共通线后期/个人线初期 |
| 第三层 | 千夏从小就被教导"要坚强，不能让爸妈担心"。父母赴海外后，她把"不麻烦别人"变成了信条。她不是不想依靠别人，而是已经不知道怎么开口说"我需要帮助"。她照亮了所有人，却从不允许自己被照亮 | 个人线中期 |

### 3.2 核心矛盾

千夏的阳光不是面具，她是真的开朗。但她的开朗有一个盲区——她把所有人的情绪都照顾到了，唯独不允许自己脆弱。从小到大，每次父母不在身边，她都告诉自己"要坚强，不能让爸妈担心"。久而久之，"坚强"变成了她的本能，"不麻烦别人"变成了她的信条。她不是不想依靠别人，而是已经不知道怎么开口说"我需要帮助"。她照亮了所有人，却从不允许自己被照亮。

### 3.3 角色色彩设计

| 项目 | 内容 |
|------|------|
| 代表色 | 天蓝色 |
| 色彩意象 | 自由、阳光、不允许脆弱 |
| 服装倾向 | 运动风，像随时要出发去冒险 |

## 4. 七瀬千紗

| 项目 | 内容 |
|------|------|
| 姓名 | 七瀬千紗（ななせ ちさ） |
| 年龄 | 16岁 / 高二 |
| 身高 | 158cm |
| 血型 | AB型 |
| 生日 | 6月12日 |
| 萌属性 | 青梅竹马/重逢的初恋/表里反差 |
| 身份 | 星见丘学园二年级，悠真的青梅竹马 |
| 家庭 | 温馨但不安定，因父亲工作调动频繁搬家 |
| 代表色 | 樱粉色 |
| 社团 | 文艺部 |
| 青春之壁 | 想留下与习惯了离开的拉扯——把过去的悠真当作珍宝保存，却不敢相信现在的悠真也能成为她的未来 |
| 特殊定位 | **官方女主** |

### 4.1 外貌

一头清秀的黑色长发，通常用樱粉色的发带扎成半扎发。脸上总是挂着温柔的微笑，但仔细看会发现那笑容里有一丝小心翼翼——像是在确认"这样笑对不对"。眼睛是深琥珀色，看着悠真的时候会微微发亮，但一旦被注意到就会立刻移开视线。身材匀称偏纤细，穿校服时会将领结系得格外整齐——这是她在每个新学校养成的习惯，"至少在第一天给人留下好印象"。

### 4.2 性格

**表面：温柔得体、善解人意的"好女孩"**

在外人面前，千紗是那种让人挑不出毛病的女孩——礼貌、体贴、成绩优秀、文艺部安静写东西的文学少女。她能自然地融入任何新环境，因为七年的辗转搬家让她学会了"快速适应"的技能。她会在第一天就记住所有同学的名字，会在别人尴尬时帮忙解围，会在集体活动中默默做最辛苦的部分。但这种"得体"不是天性，而是反复练习出来的生存本能。

**内面：粘人、笨拙、害怕失去的少女**

和悠真独处时，千紗会卸下那层"得体"的壳。她会毫无自觉地靠近悠真——走路时手臂贴着手臂，坐着时膝盖碰着膝盖，说话时不知不觉就凑到了很近。她不是故意的，只是……七年没有这样安心过了。但一旦意识到自己"太近了"，她就会猛地弹开，脸涨得通红，语无伦次地找借口——"我、我只是想看你在写什么！不是故意靠那么近的！"

她会偷偷给悠真塞便当——不是小春那种营养均衡的完美便当，而是她自己做的、形状有点歪的饭团，里面包着悠真小时候最喜欢的鲑鱼馅。她记得悠真所有的喜好，从"讨厌青椒但会为了不浪费而吃掉"到"思考时会无意识转笔，转三圈停一下"。这些记忆她一个都没忘，但重逢后她不敢直接说出来——因为如果悠真已经不记得了，那就太尴尬了，也太心酸了。

**反差萌细节**：
- 会在悠真不注意时偷偷看他，被发现后立刻低头假装在写东西，但笔根本没有动
- 紧张时会无意识揪自己的发带，有时候揪得太用力发带会散开，头发突然披下来，她会慌张地重新扎
- 悠真说冷笑话时，她会先面无表情地盯着他看两秒，然后忍不住"噗"地笑出来，笑完又假装没笑
- 偶尔会冒出小时候的口癖——"悠真くん"，一旦说出口就会立刻改口"桐生くん"，然后内心疯狂后悔

### 4.3 日常行为

**文艺部活动**：千紗在文艺部安静地写东西，但没有人知道她写的是什么。其实她写的是"旅行笔记"——记录在每一个城市见到的风景、遇到的人、吃到的食物。每一本笔记的最后一页，都写着同一句话："如果有一天回到星见市，我想带悠真くん去看这些地方。"七年的搬家生活让她变成了一个沉默的观察者，而文字是她唯一不会失去的东西。

**放学后**：千紗会找各种理由接近悠真——"文艺部要采访学生会的活动""我写了篇短文想让你看看""这家店的限定甜品只有今天有"。她不是在追求，而是在试探——试探悠真是否还像从前一样，试探自己是否还有资格靠近。

**记录习惯**：千紗有一个随身携带的小相机，拍的不是风景，而是日常的碎片——教室窗台上的阳光、走廊尽头的夕阳、操场上奔跑的背影。她害怕遗忘，所以用照片和文字把一切都留住。但她的相机里，出现最多的背影，始终是同一个人。

### 4.4 与悠真的关系

**小时候**：千紗和悠真是邻居，从小一起长大。悠真是她第一个朋友，也是她第一个喜欢的人。小时候的千紗比现在坦率得多——会拉着悠真的手跑过樱花隧道，会在他耳边说"我们永远在一起吧"，会在打雷的夜晚理直气壮地敲他的门说"我害怕"。那时候的她，不知道"永远"是一个多么奢侈的词。

**分别的七年**：小学四年级，千紗因为父亲工作调动离开了星见市。此后七年辗转了四个城市，每次刚交到朋友就要搬家。她学会了不投入太多感情，学会了笑着说"没关系，我已经习惯了"，学会了在离开时不回头——因为回头就会哭。但她始终没有学会忘记悠真。在每一个新城市的第一个夜晚，她都会翻出小时候的合影，然后在本子上写下"今天又想起你了"。

**重逢**：高二春天，千紗回到了星见市。她在樱花隧道里看到了悠真——他长高了，表情变了，说"怎样都好吧"的时候眼神是空的。千紗的心在那一刻同时被两种情绪撕扯：重逢的喜悦，和"他已经不是从前的悠真了"的恐惧。她站在樱花纷飞中，叫出了那个七年没叫过的名字——"悠真くん"。然后立刻改口："……桐生くん，好久不见。"

**现在的距离**：千紗想靠近悠真，但不知道该以什么身份。青梅竹马？那已经是七年前的事了。同学？那和其他人有什么区别？她害怕太主动会吓到悠真，又害怕不主动就会像其他人一样变成"怎样都好吧"的对象。所以她选择了最笨的方式——一点一点地、小心翼翼地靠近，像小时候慢慢靠近一只受伤的小猫。

### 4.5 三层内心深度

| 层级 | 揭示内容 | 触发条件 |
|------|----------|----------|
| 第一层 | 千紗是悠真小时候的青梅竹马，七年后重逢，温柔得体但两人之间有微妙的距离 | 共通线自然揭示 |
| 第二层 | 千紗因为频繁搬家，每次刚交到朋友就要离开，已经不敢轻易投入感情。她的"得体"不是天性，而是反复练习出来的生存本能。和悠真独处时会变得粘人笨拙，因为她只有在悠真面前才敢卸下防备 | 共通线后期/个人线初期 |
| 第三层 | 千紗一直记得悠真是她第一个喜欢的人，也是她不断回来的理由。但她害怕靠近——不是害怕悠真变了，而是害怕**自己已经配不上记忆中的位置**。七年辗转让她变成了一个"随时准备离开的人"，她不确定自己还有资格留在悠真身边。她最深的恐惧是：如果悠真重新接纳了她，下一次搬家时她又要离开——那比从未重逢更痛苦 | 个人线中期 |

### 4.6 核心矛盾

千紗的核心矛盾是**"想留下"与"习惯了离开"的拉扯**。

她比任何人都想留在悠真身边，但七年的搬家生活让她形成了一个根深蒂固的信念：**所有关系都是暂时的，所有人最终都会分开**。她不敢全力靠近，不是因为不在乎，而是因为太在乎——如果她像小时候那样毫无保留地依赖悠真，然后又不得不离开，那她承受不起第二次失去。

她把悠真当成"回忆中不可触碰的珍宝"，而不是"可以重新拥有的现实"。这就是她的青春之壁：**她把过去的悠真当作最珍贵的记忆保存着，却不敢相信现在的悠真也能成为她的未来**。

### 4.7 千紗的"拒绝"

在个人线中，当悠真终于主动靠近千紗、表达想和她在一起的意愿时，千紗会**拒绝**——不是因为她不喜欢悠真，恰恰相反，是因为她太喜欢了。

她的拒绝逻辑是："我可能又要搬走。如果我接受了你的心意然后又离开，你会比现在更受伤。我不想成为又一个让你觉得'怎样都好吧'的人。"

这个拒绝是千紗最痛苦的选择——她用伤害自己的方式来保护悠真。但千紗的"保护"是错误的——悠真需要的不是被保护，而是被选择。当悠真说出"就算你要走，我也想选择现在和你在一起"时，千紗的青春之壁才会真正突破。

### 4.8 角色色彩设计

| 项目 | 内容 |
|------|------|
| 代表色 | 樱粉色 |
| 色彩意象 | 温柔、怀旧、重逢、一触即碎的美好 |
| 服装倾向 | 柔软的针织和长裙，温暖感。但细节上会露出"随时可以离开"的痕迹——比如总是背着装有全部重要物品的包，鞋柜里永远只有一双鞋 |

### 4.9 关键台词

| 场景 | 台词 |
|------|------|
| 重逢 | "悠真——……桐生くん，好久不见。" |
| 日常靠近 | "路上喝。草莓味的，你小时候最喜欢的。" |
| 被发现偷看 | "我、我只是在看窗外的风景！不是在看你！" |
| 第二层揭示 | "你知道吗……每次搬家的时候，我都会想，如果不用和任何人说再见就好了。但那样的话，也就不会有重逢了，对吧？" |
| 拒绝悠真 | "对不起……我不能。不是因为不喜欢你——是因为太喜欢了。喜欢到如果接受了你的心意又不得不离开，我会比现在更痛苦。你也一样会痛苦的。" |
| 青春之壁突破 | "我一直以为，回忆里的悠真くん是最珍贵的，所以不敢靠近现在的你。但我错了——你一直都在这里，是我不敢相信你还在。" |
| Good End | "这次，我不会再走了。就算要走，也要牵着你的手一起走。" |

## 5. 篠崎茜

| 项目 | 内容 |
|------|------|
| 萌属性 | 灰姑娘/自卑的打工少女 |
| 身份 | 星见丘学园二年级，打工少女 |
| 家庭 | 贫穷单亲家庭，与母亲同住 |
| 代表色 | 茜色 |
| 社团 | 图书委员会 |
| 青春之壁 | 贫穷让她看不见自己的价值 |

### 5.1 三层内心深度

| 层级 | 揭示内容 | 触发条件 |
|------|----------|----------|
| 第一层 | 茜是安静的打工少女，几乎不参加任何课外活动，总是独来独往 | 共通线自然揭示 |
| 第二层 | 茜的家庭非常贫穷，单亲母亲打两份工，她自己也每天打工到很晚 | 共通线后期/个人线初期 |
| 第三层 | 茜其实成绩优秀、文章写得好，但贫穷让她觉得自己"不配"拥有梦想和友情。她用"我不需要"来回避"我想要但得不到"的痛苦 | 个人线中期 |

### 5.2 核心矛盾

茜不是没有才华——她成绩优秀，文章写得好，对文学有深刻的理解力。但贫穷让她觉得自己不配拥有梦想。她不敢和同学走得太近，因为害怕被看到自己穿旧衣服、吃最便宜的便当。她把所有的自卑藏在安静的外表下，用"我不需要"来回避"我想要但得不到"的痛苦。

## 6. 天羽小春

| 项目 | 内容 |
|------|------|
| 萌属性 | 嫁力超群的义妹/温柔理性的暗中守护者 |
| 身份 | 星见丘学园一年级，悠真的义妹 |
| 家庭 | 继父之女，悠真的义妹，与悠真无血缘关系 |
| 代表色 | 铃兰白 |
| 社团 | 美术部 |
| 青春之壁 | 最了解他的人，最不能靠近他——理性选择了不告白，但理性无法消灭感情 |

### 6.1 三层内心深度

| 层级 | 揭示内容 | 触发条件 |
|------|----------|----------|
| 第一层 | 小春是嫁力超群的义妹，家务全能，对哥哥体贴入微。看起来只是"好妹妹" | 共通线自然揭示 |
| 第二层 | 小春对悠真的了解远超普通兄妹——她知道他的一切习惯、情绪、逃避方式。她画的都是悠真的背影，因为"正面……我会心跳加速"。和悠真独处时会变得坦率，会吐槽、会天然呆，这种反差只有悠真能看到 | 共通线后期/个人线初期 |
| 第三层 | 小春的理性是她最大的牢笼。她比任何人都清楚"不应该喜欢哥哥"——重组家庭的微妙、社会的眼光、可能失去一切的后果。她不是不敢告白，而是理性地选择了不告白。但理性无法消灭感情，只会让感情在沉默中发酵成痛苦。她最害怕的不是被拒绝，而是告白之后连"妹妹"的位置都保不住 | 个人线中期 |

### 6.2 小春与悠真的关系

小春在母亲去世后跟随父亲。父亲再婚后，她有了新的母亲和——一个哥哥。从第一天起，悠真就对她很好：帮她搬行李、带她认路、在她害怕打雷的夜晚安慰她。小春很快发现，自己对悠真的感情不只是"哥哥"。但"兄妹"这个身份像一道无形的墙——她不敢越界，也害怕越界后会失去现在拥有的一切。

### 6.3 小春的日常

小春是桐生家真正的"管家"。每天比悠真早起半小时做早餐，校服永远熨得平整，便当营养均衡且会根据悠真前一天的状态调整菜单。她不是被迫做这些——"照顾哥哥"是她最幸福的时刻。悠真的每一个习惯她都记在心里：思考时无意识转笔、紧张时摸左手腕、说"怎样都好吧"的时候其实最在意。她比悠真自己更了解悠真。

### 6.4 小春的双面性

在家人面前，小春是安静乖巧的妹妹，话不多，总是微笑着做自己的事。但和悠真独处时，她会变得坦率——悠真说冷笑话时她会面无表情地吐槽，偶尔冒出让人哭笑不得的天然发言。这种反差是只有悠真才能看到的。在其他角色的路线中，小春是那个能冷静指出悠真问题的人——"哥哥，你在逃避吧""你其实很想靠近她，对吗？"。她用自己的理性帮助哥哥，即使这意味着把他推向别人。

### 6.5 核心矛盾

小春的理性是她最大的牢笼。她比任何人都清楚"不应该喜欢哥哥"——重组家庭的微妙、社会的眼光、可能失去一切的后果。她不是不敢告白，而是理性地选择了不告白。但理性无法消灭感情，只会让感情在沉默中发酵成痛苦。她最害怕的不是被拒绝，而是告白之后连"妹妹"的位置都保不住。

| 色彩意象 | 纯净、理性、隐藏的心意 |
| 服装倾向 | 素色围裙配居家服，出门简洁棉质，不加纹饰但讲究质感 |
