# 《君色の季節》— 好感度系统

> 对应工作流：**二、角色设计阶段 → 2.1 角色企划 → 好感度系统**

---

## 1. 好感度系统

### 1.1 基本参数

- 每个可攻略角色独立好感度，范围 0-100
- 好感度通过日常选择、约会事件、关键对话累积
- 好感度影响：日常对话变化、额外事件触发、结局走向

### 1.2 好感度区间

| 好感度区间 | 影响 |
|-----------|------|
| 0-19 | 角色对悠真保持礼貌距离 |
| 20-39 | 角色开始对悠真敞开心扉，日常对话增加 |
| 40-59 | 角色主动找悠真，触发专属日常事件 |
| 60-74 | 进入好感区间，角色对悠真产生明确好感 |
| 75-100 | 进入告白区间，解锁Good End条件 |

---

## 2. 依赖度系统

依赖度是隐藏参数，反映角色对悠真的情感依赖程度。

### 2.1 依赖度区间

| 依赖度区间 | 表现 |
|-----------|------|
| 0-29 | 角色可以独立面对问题 |
| 30-49 | 角色开始依赖悠真的意见 |
| 50-59 | 角色在关键时刻会寻求悠真的帮助 |
| 60-100 | 角色对悠真产生深度情感依赖，解锁关键剧情 |

### 2.2 依赖度与好感度的区别

| 维度 | 好感度 | 依赖度 |
|------|--------|--------|
| 性质 | 角色对悠真的好感程度（表层） | 角色对悠真的情感依赖程度（深层） |
| 可见性 | 玩家可通过对话和事件推断 | 完全隐藏，不显示 |
| 影响 | 日常对话、事件触发 | 关键分支、结局质量 |
| 累积方式 | 正面选择累积 | 深层选择（倾听、陪伴、理解）累积 |

### 2.3 好感度+依赖度与结局

| 好感度 | 依赖度 | 结局类型 |
|--------|--------|----------|
| 高 | 高 | Good End（恋爱结局） |
| 高 | 低 | 友情结局（未达成Good End但非Bad End） |
| 低 | 高 | Bad End（依赖但不信任） |
| 低 | 低 | 陌生人结局（无交集） |

## 3. 悠真成长系统

### 3.1 成长度变量

`yuma_growth`（0-100），影响内心独白内容、日常对话态度、True End解锁。

### 3.2 成长阶段

| 成长阶段 | 触发条件 | 悠真变化 | 对应台词变化 |
|----------|----------|----------|-------------|
| 局外人（初始） | 游戏开始 | 旁观者心态，在家庭中感到疏离 | "怎样都好吧。" |
| 微光 | 好感度≥20（任一角色） | 开始在意某个人的事 | "她……好像在勉强自己。" |
| 着色 | 好感度≥40（任一角色） | 主动想要帮助对方 | "我想为她做点什么。" |
| 归属 | 好感度≥60，进入个人线 | 感受到被需要的重量 | "原来被需要是这种感觉。" |
| 觉醒 | 个人线第六章关键选择 | 做出勇敢的选择，不再逃避 | "这次，我不会再退缩了。" |
| 有色 | Good End达成 | 找到自己的归属和意义 | "我不再是局外人了。" |

### 3.3 成长子维度

| 子维度 | 变量名 | 说明 |
|--------|--------|------|
| 勇气 | `yuma_courage` | 面对困难的勇气，影响关键选择是否出现 |
| 归属 | `yuma_belonging` | 感受被需要的能力，影响内心独白深度 |
