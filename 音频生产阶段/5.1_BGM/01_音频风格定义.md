# 《君色の季节》— 音频风格定义

> 对应工作流：**五、音频生产阶段 → 5.1 BGM → 音频风格定义**

---

## 1. BGM风格定义

| 维度 | 方向 |
|------|------|
| BGM整体风格 | 钢琴+木吉他为主，弦乐点缀，温暖治愈 |
| 日常BGM | 轻快钢琴曲、木吉他弹唱风 |
| 恋爱BGM | 弦乐+钢琴，甜蜜渐进 |
| 感伤BGM | 独奏钢琴、大提琴，留白多 |
| 季节BGM | 春-木吉他、夏-明亮节奏、秋-弦乐、冬-钢琴 |

## 2. 音效重点

| 类别 | 音效 |
|------|------|
| 校园 | 铃声、上课铃、走廊脚步声、教室嘈杂声 |
| 自然 | 蝉鸣、风铃、雨声、雷声、樱花飘落声 |
| 日常 | 便当盒打开声、筷子声、门铃、键盘声 |
| 情感 | 心跳声、呼吸声、沉默 |
