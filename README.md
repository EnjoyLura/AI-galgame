## 核心设计理念

> **重组家庭的男主桐生悠真**，在"家"中找不到归属——不是不被爱，而是不知道自己该站在哪个位置。**5位各有青春烦恼的少女**，每人面对自己的"青春之壁"。男主选择深入接触感兴趣的女主，了解她们的内心，帮助她们解决问题——**在救赎她们的过程中，也救赎了自己，最终找到归属**。

### 男主：桐生悠真

| 项目 | 内容 |
|------|------|
| 身份 | 星见丘学园二年级，重组家庭的长子 |
| 性格 | 温和但疏离，习惯旁观，不主动靠近任何人 |
| 困境 | 在"家"中找不到归属 |
| 核心台词 | "怎样都好吧。" |

### 五位女主角

| 角色 | 定位 | 代表色 |
|------|------|--------|
| 白鸟绫音 | 被爱束缚的温室大小姐 | 藤紫色 |
| 星野千夏 | 不允许自己脆弱的向日葵少女 | 天蓝色 |
| 七瀬千纱 | 阔别七年重逢的初恋青梅竹马（官方女主） | 樱粉色 |
| 筿崎茜 | 独立是她的骄傲，也是她不敢依赖别人的理由 | 茜色 |
| 天羽小春 | 最了解他却最不能靠近他的嫁力义妹 | 铃兰白 |

> 角色详细设定见 → [角色概念设定](1.2_角色设计/1.2.1_角色企划/01_角色概念设定.md)
